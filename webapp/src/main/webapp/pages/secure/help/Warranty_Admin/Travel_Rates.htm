Travel Rates

The repair/replacement labor may require the laborer to travel, either to an end customer's location or to purchase parts. The Dealer can claim the expenses occurred due to such a travel while submitting the claim. As an Admin, you can assign rates per hour or per distance travelled. You can also assign rates per trips made.

 

The Travel Rates option allows you to define travel rate rules based on the following four parameters: Dealer name, type of claim, warranty type and product. You can set a travel rate for each unique combination of these parameters. The rates can be applied over a specified time period, defined by the repair date range. You can create, search and maintain travel rate configurations in an easy and efficient manner using the Travel Rates option.

 

To view the travel rates page,

  • Click on the Travel Rates link under the Common Actions on the Home page.

  • Click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Flat Rates folder. All links under this folder are listed. Click on the Travel Rates link.

Either of the above actions will take you to the Travel Rates page. On the right-hand pane, you can see the Travel Rates table that lists all configurations created by you. Click Lists to see for more information on navigating through the table.

 

The Travel Rates table provides the following information about each configuration:

  • Dealer/Dealer Group: Name of the dealer or the dealer group to which this configuration is applicable.

  • Claim Type: Type of claim to which this configuration is applicable.

  • Warranty Type: Warranty type to which this configuration is applicable.

  • Product: Products to which this configuration is applicable.

To view all details in a single page, double click on the configuration in the table. The Edit Travel Rate <n> page is shown.

 

The details in the detail page are shown under various sections, for ease of navigation. The following sections are shown:

  • Conditions: You can view the following details here - dealer name, type of claim, warranty type and the products to which the rules in this configuration are applicable.

  • Rates: You can view the following details here - repair date range, in the user specified format, and the rates per hour, per distance and per trips, applicable to this date range.

You can create a new travel rate or edit an existing rate on the Travel Rates page.

 

Creating a Travel Rate

You can create a new travel rate configuration by specifying a unique set of values to the above mentioned parameters.

 

To create a new configuration:

Click on the option.

You will see the New Travel Rate page. This page is divided into various sections.

 

In the Conditions section, specify the following details:

  • Dealer: Name of the dealer to whom the new rates are applicable. You can associate the new rate with a group of dealers by clicking on the Specify Dealer Group link. The Dealer field now changes to Dealer Group field and you can enter a group name here. To switch to the Dealer field, click on the Specify Dealer link. This field supports auto-complete.

  • Product: Name of the product to which the new rates are applicable. This field supports auto-complete.

  • Warranty Type: Use this drop down option to specify the type of warranty to which the rates are applicable. The options available in this field are: All, Standard, Extended and Goodwill. You an select a particular warranty type or select All to apply the rates to all warranty types.

  • Claim Type: Use this drop down option to specify various claim types to which the new rates are applicable. The options available in this field are: All, Parts, Machine, and Campaign.  You can select a particular claim type or select All to apply the rates to all claim types.

In the Rates section, you can specify rules to apply travel rates for specific repair date ranges:

  • Click on icon. This adds a row allowing you to specify the repair date range and the rate.

  • Repair Date: Use the option in the From and To fields to specify the start of the potential date range. The date will be displayed in the format selected by you.

  • Travel Rate: You can specify the travel rate on the following basis: per hour, per unit distance covered, and per trips made. Each criterion can be used independent of the other.

  • You can add more rows and set more rules for different date ranges.

  • To delete a rule, click on icon corresponding to that row.

  • If you to close this page without saving any data entered by you, click Cancel.

  • Click Save to save the new rate. A success message is shown indicating the new configuration is saved.

Note: When creating rules for a configuration, you need to conform to the following:

  • To date cannot be before the From date.

  • From date can be a value placed in past, present or future.

  • To date should be a value placed in future.

  • If more than one rules are created, the date difference in the To date of one rule and the From date of the consecutive rule should be a single day.

  • You cannot assign overlapping values to From and To date fields.

Editing a Travel Rate

You can edit a travel rate configuration and reset the parameters to suit your needs.

 

To edit a configuration:

  • Double click on a configuration to open it in the detailed page.

  • You can edit any field in this page.

  • Click Cancel if you want to close this page without saving any changes made by you.

  • Click Save to save the edited rate. A success message will be shown indicating all changes are saved.

  • To delete this configuration, click Delete . A success message will be shown indicating the configuration is deleted.

Note: You can edit the To date to a value in the past.
