LIST MODELS

Every product has various models and each model is identified by a model name or number.

 

To view a list of models present in the system, click on List Models tab under the Warranty Admin tab on the left-hand pane. You can group a set of models and create a label.

 

To create and apply a label,

  • Select the model to which you want to apply a label. You can select multiple models by pressing the Ctrl key and simultaneously clicking on the models.

  • Click on the  button to apply a label to the selected model. A pop-up window appears. Select Create.

  • In the Label field, enter the name of the label that you want to create.

  • Click Select. The label name that you entered will be listed in the Selected Labels section.

  • To apply the selected label, click Apply Label(s). The selected label will be applied to the selected models.

The icon refers that no label has been applied. When a label is applied, the icon changes to . If you place your mouse cursor on this icon, the names of label applied will be displayed.
