cREATE/UPDATE ATTRIBUTES

An attribute is a property of the inventory item used while filing a claim. If you want to give some additional information about the part/inventory while filing a claim, you can associate an attribute to that part.

 

Perform the following steps to create an attribute:

  • Click on Additional Attributes tab under the Warranty Admin tab on the left-hand pane. Click on Create/Update Attributes tab under Additional Attributes. A Manage Attributes page will open on the right-hand side.

  • Click on button in the listing toolbar.

  • Enter the following details under the Attribute Details page:

  • Attribute Name: Enter the attribute name in this field.

  • Claim Type: Select the claim type from the drop down. You can also select multiple claim types by pressing Ctrl key and clicking on the claim types.

  • Data Type: Select the data type from the drop down. A data type can be number, text or date. If the data type doesn’t match while entering information to this attribute during claim submission, you will receive an invalid data type error.

  • Purpose: Select the purpose for which you are creating an attribute. An attribute can be created either for ‘Part Source’ or ‘Claimed Inventory’.

  • Mandatory: There are two options. Select ‘Yes’ if you do not want the attribute field to be left blank while filing a claim. You will receive an error while validating the claim if it is left blank. If you select ‘No’, you have the option to leave the attribute field blank.

  • Click Submit. A success message will be shown confirming your action.

The newly created attribute will now appear under the attribute list in the Manage Attributes page. You can now associate an attribute to any part or inventory. Click on the below links to know more about association and dissociation of part or inventory attributes.

  • Associate/Dissociate Part Attributes

  • Associate/Dissociate Inventory Attributes

You can also update the already created attribute. To update, click on the attribute and edit the Claim Type or Mandatory selection. All the other fields are not editable. Perform one of the following actions:

  • Submit: If you have updated the attribute, then click this button. The attribute will be updated with your new selections.

  • Delete: If you do not want the attribute any longer, then click this button. This will remove the attribute from the list.

  • Cancel: Click on this button to disregard the action. It will move you to the previous page.
