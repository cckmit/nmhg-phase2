VIEW Labels

Labeling is a way of grouping objects together in a flat structure. This also referred to as tagging of objects. The objects under a label are homogenous, i.e. a certain label can have objects of the same kind under it. These labels can be used while creating various business rules.

 

Perform the following steps to search a label:

  • Click on View Labels tab under the Warranty Admin tab on the left-hand pane.

  • Select a category for the label.

  • Click Submit.

You can select a label from a list of labels displayed on the View labels page. It will give you a list of all the inventories or policies or job codes tagged under that specific label.

 

Applying Labels

You can apply a label to an item or a group of items.

 

To apply a label,

  • Select the item to which you want to apply a label. You can select multiple items by pressing the Ctrl key and simultaneously clicking on the products.

  • Click on the  button to apply a label to the selected item. A pop-up window appears. Select Create.

  • In the Label field, enter the name of the label that you wish to apply. This field supports auto-suggest for all previously created labels. You can also create a new label by entering its name in this field.

  • Click Select. The label name that you entered will be listed in the Selected Labels section.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To apply the selected label, click Apply Label(s). The selected label will be applied to the selected items.

The icon refers that no label has been applied. When a label is applied, the icon changes to . If you place your mouse cursor on this icon, the names of label applied will be displayed.

 

Removing labels

You can remove an already applied label from an item or a group of items.

 

To remove a label,

  • Select the items from which you wish to remove the applied labels. You can select multiple items by pressing the Ctrl key and simultaneously clicking on the items.

  • Click on the button to remove a label applied on the selected items. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to remove. This field supports auto-suggest for all previously created labels.

  • Click Select. The label name that you entered will be listed in the Selected Labels section.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To remove the selected labels, click Remove Label(s). The selected labels will be removed from the selected items.
