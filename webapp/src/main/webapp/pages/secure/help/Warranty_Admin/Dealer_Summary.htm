DEALER SUMMARY

There are various attributes set up for a dealer such as travel rates, labor rates and modifiers. The Dealer Summary page gathers and displays all the details for a dealership on one page.

 

Click on Dealer Summary under the Dealer Information tab on the left hand pane to view the dealer details. The page is divided into various sections.

 

Dealer Summary

This section displays the Dealership Name, Dealership Number, Dealership Type and the Certified or Non-certified status of the dealer.

Network Structure

This section displays the Business Unit, Region and the District to which the dealership belongs.

Labor Rates

This section displays the assigned labor rates for a particular period. The details include the Product type, Claim Type, Warranty Type, Customer Type, Base Rate, Date range and the Business Unit.

Travel Rates

This section displays the assigned travel rates such as Distance Rate, Hourly Rate and Trip Rates for a particular period.

Modifiers

This section displays all the modifiers created along with the modifier value for a particular period. Payment modifiers allow you to alter the claim amount either through discounts or surcharges for each of these components.
