Truck Details

This is the detailed page of a truck which displays the complete information about a given inventory item. It includes information about the product, the complete service history and ownership history of the inventory item.

 

You can get to this page by clicking on the truck serial number displayed anywhere in the Warranty Management application, except on the truck details page. You can also view the truck details by querying for an inventory item. For more details on querying an inventory item, click here.

 

The data about a truck is displayed under various sections on the truck details page for ease of navigation. The fields under some sections on this page may be hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses it, hiding all fields under it.

 

Truck Info

You can view the following information about the selected truck, in this section:

  • Serial Number: Serial number of the truck.

  • Business Unit Name: The name of the Business Unit for this truck.

  • Product Type: Product type to which this truck belongs.

  • Manufacturing Site: The manufacturing location of the truck.

  • Product: The name of the product to which this truck belongs.

  • Model Number: Model number of the product to which this truck belongs. This model number is displayed as an hyperlink. Click on this link to view the specific assembly BOM/diagram for this truck.

  • Shipment Date: Date when the truck was shipped.

  • Item Number: This is the identification number of the item to which this truck belongs. This number determines the unique class of product of which type this truck is.

  • Item Description: A short description of the item.

  • Pre-Order Booking: This flag displays Yes only if the dealer has ordered the inventory by specifying customer details.

  • Unit Hours/Energy Units: Usage on the truck as updated from the last claim submitted on this truck.

  • Warranty Start Date: The warranty start date of the truck.

  • Warranty End Date: The warranty end date of the truck. The warranty end date is calculated as Warranty Start date + Number of months covered from the date of delivery/shipment.

  • Item Condition:  The current condition of the item - scrap, unscrap,  new or previously owned.

  • Sales Order Number: Sales order number associated with the truck.

  • Ownership: The type of ownership is shown here.

  • Invoice: Invoice number associated with the transaction of this truck from the OEM to the owning service provider.

  • Data Source: Displays the data source of the truck.

Owning Dealer

The name of the dealer who owns this truck in his/her stock or who retailed this truck to an end customer, will be displayed in this field. The following details about the owning dealer are shown in this section:

  • Dealer Number: Identification number of the service provider.

  • Dealer Name: Name of the service provider.

  • Dealer Address: Primary mailing address of the service provider.

  • Dealer City: City of the service provider.

  • Dealer State: State of the service provider.

  • Dealer ZIP: ZIP code of the service provider's location.

  • Dealer Country: Country of the service provider.

Owner Information

This information is shown if the truck is in the retailed state 'OR' Pre-Order Booking is Yes  for the truck.

The name of the customer who purchased this truck from the dealer, will be shown here. The name will be a hyperlink to the Owner Details pop-up window. The following details are shown in the Owner Details pop-up window about the current owner of this truck:

  • Owner Number: Identification number of the end customer owning this truck.

  • Owner Name: Name of the end customer.

  • Owner  Address: Primary mailing address of the end customer.

  • Owner City: City of the end customer.

  • Owner State: State of the end customer.

  • Owner ZIP: ZIP code of the location of the end customer.

  • Owner Country: Country of the end customer.

Major Components

This section displays all the major components associated with the truck.

  • Serial Number: Serial Number of the major component. The serial number if a hyperlink to the truck detail page.

  • Part Number: Part number of the component.

  • Description: A short description of the component auto-populates.

  • Install Date: Date on which the component was installed.

  • Warranty Start Date: Date when this component was covered by the warranty policies associated with it.

  • Warranty End Date: Date when this component will be out of warranty, i.e. the policies covering this component will be inactivated.

Transaction History

This section displays all the ownership transaction changes that have happened on this truck. For each transaction the following details are shown:

  • Transaction Date: The date when the transaction happened.

  • From Company: The seller in the transaction.

  • Move to Customer: The buyer in the transaction. The last transaction shows the current owner of this truck in this field.

  • Customer Type: The type of the buyer. The buyer maybe of type: dealer, end customer, national account, government account, consignment account, etc.

  • Transaction Type: The type of transaction. The transaction may be Delivery Report (DR), truck transfer (TTR), etc.

  • Invoice: Invoice number of the invoice attached as a proof of expense while filing the claim.

Recovery Claim History

This section appears only for recovery claims. It displays the details of all recovery claims that have been filed so far for this truck.

  • Recovery Claim state: Current status of the recovery claim.

  • User: Login details of the user for this recovery claim.

  • Comments: Comments mentioned while filing a recovery claim.

  • Accepted/Rejected/Cannot Recover Reason: Reason associated with the acceptance, rejection or cannot be recovered for the claim.

  • Recovered Amount: Amount to be recovered under the claim.

  • Date: Date on which the recovery claim was filed.

Claim History

This section shows details of all claims that have been filed so far on this truck. For each claim, the following details are shown:

  • Claim Number: The six-digit claim number associated with the claim. If this claim was a campaign claim, then you can see the campaign code along with the claim number in this field. The claim number if a hyperlink to the claim detail page.

  • Date of Claim: Date on which the claim was filed.

  • Status: Current status of this claim.

  • Failure Date: Date of failure as mentioned in this claim.

  • Parts Replaced: Parts replaced during the repairing/servicing of the truck.

  • Hours in Service/Energy Units: Number of hours the truck was in service till the date of failure.

  • Fault Location: Fault location as mentioned in the claim.

If you were the dealer who filed a claim on this truck, then you can also see the following payment details:

  • Total Amount Asked: Total claim amount asked by you in this claim.

  • Total Amount Paid: Total claim amount paid by the OEM for your claim.

Field Product Improvement

This section displays any outstanding service campaigns on this truck. For each campaign the following details are displayed:

  • Field Product Improvement Code: Campaign code associated with the campaign. This will be a hyperlink to the campaign detail page.

  • Class: Kind of campaign selected for the truck.

  • End Date: Date when this campaign will be deactivated.

  • Description: A short description of the campaign.

  • Status: Active or Inactive status of the campaign.

Warranty Coverages

You can view a list of all parts and machine policies associated with this truck. The policies listed here include all standard, extended and goodwill policies registered on this truck. Some of these policies may be active and others may be deactivated/expired.

 

The policy details are displayed in a tabular format, with the following information being shown for each policy:

  • Policy Code: Code assigned to the policy. This code uniquely determines a warranty policy. Clicking on this link will take you to the policy details page for this policy. For further information on the policy details page, click here.

  • Policy Name: Name of the policy.

  • Type: Kind of policy. A policy can be of type standard, extended or goodwill.

  • Start Date: Date when this policy will start providing warranty coverage for this truck.

  • End Date: Date when the warranty coverage on this truck, provided by this policy, will expire. The warranty end date is calculated as Warranty Start date + Number of months covered from the date of delivery/shipment.

  • Months Covered: Number of months of warranty coverage provided by this policy.

  • Hours Covered: Number of hours of warranty coverage provided by this policy.

  • Status: Current status of the warranty coverage.

  • Comments: Comments mentioned while registering the policy.

Actions

Depending on the state of the truck, certain actions are allowed on the truck details page.

  • Truck Transfer: Clicking on this button will open the Truck Transfer page with details of this truck populated in it.

  • Retail Truck Transfer: Clicking on this button will open the Retail Truck Transfer page with details of this truck populated in it.

  • Extended Warranty Purchase: Clicking on this button will open the Extended Warranty Purchase page with details of this truck populated in it.

  • Mark as Scrapped: Clicking on this button will open the Scrap Inventories page with the serial number of this truck populated in it.

  • Create Claim: Clicking on this page will open the claim submission page with details of this truck populated in it. F

  • Reports: Clicking on this page will open the Custom Reports page. You can fill these reports and submit it for this truck.
