DUE PARTS RECEIPT

All shipments sent to the warehouse are listed in your Due Part Receipt folder. You can verify if the parts mentioned in the shipment have been physically received and also inform the dealer about any missing parts.

 

To view the parts that has been shipped to this warehouse, click on the Part Returns tab on the left-hand pane. Click on the Due Part Receipt folder. On this page you can see the Due Parts Receipt table that lists all shipments sent to this warehouse. For further information on navigating through the table, see Lists.

If you double click on any entry in this table, all details relevant to that shipment will be displayed on a new page, Shipment Number <n>.

 

You can view/edit the following details for each part, in the Claim Number <n> page:

  • Claim Details: The following details are displayed for a claim whose part is shipped in this shipment:

  • Claim Number: Identification number of the claim which cites this part as a replaced component. Click on the claim number link to view claim details.

  • Serial Number: Serial number of the truck to which this part belongs. Click on the serial number link to view truck details.

  • Model: Model number of the truck to which this part belongs.

It also provide details about the shipment sent to this warehouse.

  • Return to Location: The warehouse location where the shipment is sent.

  • Dealer: Name of dealer who sent this shipment.

  • Carrier: Name of the carrier in which the consignment was shipped.

  • Tracking Number: Dealer's internal tracking number for this consignment.

  • Shipment Date: Date when the dealer made this shipment.

  • Part Details: The following details are shown for each part shipped in this shipment:

  • Serial Number: Serial number of the part.

  • Part Number: Part number of the part.

  • Description: A short description about the part.

  • Returned Part Serial Number: Serial number of the returned part.

  • Returned Part Number: Part number of the returned part.

  • Returned Part Description: A short description about the returned part.

  • Return Location: Use this drop down option to select an aisle in the warehouse where this part must be placed, to be picked up for further processing. This drop-down lists all aisles in this warehouse.

  • Receipt Status: Use this drop-down field to specify if or not you have received a part shipped by the dealer in this shipment. If you have received a part, as mentioned in the shipment, then select the option Received in this drop-down field. If the part is not received in this shipment, then select the option Not Received in this drop-down field.

If you mark a part as Not Received, then a notification is sent to the dealer's Due Parts or Overdue Parts folders depending on the due date for the part.

  • Comments: If you have any additional comments on this part, you can write the same in this text field.

These details are repeated for each part in the shipment. You need to select a location and an action for each part before taking an action on the entire shipment. Click Submit to submit the actions taken by you on all parts included in this shipment.

All parts marked as not received will be sent back to the dealer's due or overdue folders. A success message will be shown confirming your action. The message will hide itself and the current page will be closed.
