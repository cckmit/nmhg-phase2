SEARCH PARTS

The Warranty Management application allows you to search for replaced parts returned by any service provider to any location, as well as parts that are yet to be returned to a warehouse, by using the Search Parts option.

 

To go to the search page, click on the Part Returns tab on the left-hand pane. The tab will expand to show all folders under it. Click on the Define Search Query link. You can search for equipments based on various part related parameters on this page. You can also save a search for future viewing.

 

You can search for positive results for a set of conditions that you define on this page, or you can also search for a negation of the set of conditions defined on this page. To know more about creating and editing existing business conditions, click here.

 

Search Results

The results of your search will be shown in a tabular format. Click here to know more about the listing toolbar on the Search Results table. Click here to know more about navigating through the Search Results table. To view the details on a single page, double click on that entry in the table. The page is divided into two sections, Claim Details and Part Details for ease of navigation.

 

Edit Search

You can edit a search phrase and modify any of the search parameters.

 

To edit a search phrase,

  • Click on the button on the search result page.

  • The Edit Search Query page is shown. You can see the search conditions created by you here.

  • You can edit any parameter in this field.

  • To reset the data in this page, click Reset. All details entered by you will be reset to the default selections.

  • To execute the modified search conditions, click Execute. The search results will be shown in a tabular format.

Search Parts Using PreDefined Search

You can also search the part using the PreDefined Search options listed under the Part Returns tab on the left-hand pane or click on the Predefined Search (Part Return) link under the Common Actions on the Home page. To search for a part, enter/select one or more parameters.

  • Dealer Name: The name of the dealer.

  • Dealer Number: The unique number assigned to the dealer.

  • Claim Number: The number of the claim to which this part belongs.

  • Serial Number: Serial number of the machine or the part, depending on the claim type, on which you filed the claim.

  • Barcode Number: A barcode number which uniquely identifies the part.

  • Tracking Number: Tracking number of the part returned.

  • Status: Select the status of the part from the drop down list.

  • Return To Location: Location of the warehouse where the part is returned.

Once you set the search parameter, perform one of the following actions:

  • If you wish to save this search, then check the Save Search check-box. The search will be saved in the Search Folders section under the search phrase specified by you.

  • If you want to clear the options selected by you, click on the Reset button.

  • If you want to search for a part based on the condition you just created, click on the Search button. The page refreshes to show the search results in a tabular format.
