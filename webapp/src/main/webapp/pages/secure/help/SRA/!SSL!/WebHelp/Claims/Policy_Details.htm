POLICY DETAILS

The policy details page is shown when you click on the policy code, anywhere in the Warranty Management application. This page displays all details about the policy's coverage and applicability.

 

The policy details page is a read-only page, divided into various sections for ease of navigation. You can see the following sections here: Policy Details, Applicability Terms and Registration Terms. The fields under some of these sections may be hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses, it hiding all fields under it.

 

Policy Details

You can view the following details in this section:

  • Policy Code: Identification code of the policy.

  • Policy Name: Name of the policy.

  • Warranty Type: Kind of warranty provided by the policy. The warranty type can be standard, extended or goodwill.

  • Months Covered: Number of months of coverage offered by this policy.

  • Active From and Active Till: Date range between which this policy is available for registration.

  • Priority: The priority assigned to this policy.

  • Comments: Any comments provided by the administrator while setting up this policy.

Applicability Terms

This section lists all applicability terms that must be satisfied for this policy to be picked up for a warranted claim.

 

Registration Terms

You can view the following details in this section:

  • Registration Price: Cost of this policy. If the registration price is zero, this policy is available by default.

  • Transferrable: Indicates if this policy is transferrable to a new end user during equipment transfer.

  • Transfer Fee: If the equipment is transferrable, then this field indicates the fee to be paid for transferring this policy to the new end user.

  • Products Covered: A list of products covered by this policy.

  • Dealers Covered: A list of dealer groups/dealer availing this policy. If this is empty, then this policy is applicable for all dealers.
