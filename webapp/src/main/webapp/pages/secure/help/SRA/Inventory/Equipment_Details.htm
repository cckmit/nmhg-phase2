EQUIPMENT DETAILS

The equipment details preview pane and detail page display the complete information about a given inventory item. It includes information about the product, the complete service history and ownership history of the inventory item.

 

You can get to this page by clicking on the equipment serial number displayed anywhere in the Warranty Management application, except on the equipment details page. You can also view the equipment details by querying for an inventory item. For more details on querying an inventory item, click here.

 

The data about an equipment is displayed under various sections on the equipment details page for ease of navigation. The fields under some sections on this page may be hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses it hiding all fields under it.

 

Equipment Info

You can view the following information about the selected equipment, in this section:

  • Serial Number: Serial number of the equipment.

  • Product: Product to which this equipment belongs.

  • Model: Model number of the product to which this equipment belongs.

  • Manufacturing Site: The manufacturing location of the equipment.

  • Item Number: This is the identification number of the item to which this equipment belongs. This number determines the unique class of product of which type this equipment is.

  • Item Description: A short description of the item.

  • Hours in Service: Usage on the equipment as updated from the last claim submitted on this equipment.

  • Shipment Date: Date when the equipment was shipped.

  • Item Condition: The current condition of the item - scrap or unscrap.

  • Invoice Number: Invoice number associated with the transaction of this equipment from the OEM to the owning service provider.

  • Sales Order Number: Sales order number associated with the equipment.

Dealer Details

The name of the dealer who owns this equipment in his/her stock or who retailed this equipment to an end customer, will be displayed in this field. This will be a hyperlink to the Dealer Details pop-up window. The following details about the owning dealer are shown in the Dealer Details pop-up window:

  • Dealer Number: Identification number of the service provider.

  • Dealer Name: Name of the service provider.

  • Dealer Address: Primary mailing address of the service provider.

  • Dealer City: City of the service provider.

  • Dealer State: State of the service provider.

  • Dealer ZIP: ZIP code of the service provider's location.

  • Dealer Country: Country of the service provider.

Owner Details

 This information is shown only if the equipment is in the retailed state.

 

The name of the customer who purchased this equipment from the dealer, will be shown here. The name will be a hyperlink to the Owner Details pop-up window. The following details are shown in the Owner Details pop-up window about the current owner of this equipment:

  • Owner Number: Identification number of the end customer owning this equipment.

  • Owner Name: Name of the end customer.

  • Owner  Address: Primary mailing address of the end customer.

  • Owner City: City of the end customer.

  • Owner State: State of the end customer.

  • Owner ZIP: ZIP code of the location of the end customer.

  • Owner Country: Country of the end customer.

Major Components

This section displays all the major components associated with the equipment.

  • Serial Number: Serial Number of the major component. The serial number if a hyperlink to the equipment detail page.

  • Part Number: Part number of the component.

  • Description: A short description of the component auto-populates.

  • Install Date: Date on which the component was installed.

  • Warranty Start Date: Date when this component was covered by the warranty policies associated with it.

  • Warranty End Date: Date when this component will be out of warranty, i.e. the policies covering this component will be inactivated.

Transaction History

This section displays all the ownership transaction changes that have happened on this equipment. For each transaction the following details are shown:

  • Transaction Date: The date when the transaction happened.

  • From Company: The seller in the transaction.

  • Move to Customer: The buyer in the transaction. The last transaction shows the current owner of this equipment in this field.

  • Customer Type: The type of the buyer. The buyer maybe of type: dealer, end customer, national account, government account, consignment account, etc.

  • Transaction Type: The type of transaction. The transaction may be warranty registration (WR), equipment transfer (ETR), etc.

  • Invoice: Invoice number of the invoice attached as a proof of expense while filing the claim.

Recovery Claim History

This section shows details of all recovery claims that have been filed so far for this equipment.

  • Recovery Claim State: Current status of the recovery claim.

  • Comments: Comments mentioned while filing a recovery claim.

  • Accepted/Rejected Reason: Reason associated with the acceptance or rejection of the claim.

  • Recovered Amount: Amount to be recovered under the claim.

  • Date: Date on which the recovery claim was filed.

Claim History

This section shows details of all claims that have been filed so far on this equipment. For each claim, the following details are shown:

  • Claim Number: The six-digit claim number associated with the claim. If this claim was a campaign claim, then you can see the campaign code along with the claim number in this field. The claim number if a hyperlink to the claim detail page.

  • Date of Claim: Date on which the claim was filed.

  • Status: Current status of this claim.

  • Failure Date: Date of failure as mentioned in this claim.

  • Parts Replaced: Parts replaced during the repairing/servicing of the equipment.

  • Hours in Service/Energy Units: Number of hours the equipment was in service till the date of failure.

  • Fault Location: Fault location as mentioned in the claim.

Field Modification

This section displays any outstanding service campaigns on this equipment. For each campaign the following details are displayed:

  • Field Modification Code: Campaign code associated with the campaign. This will be a hyperlink to the campaign detail page.

  • Class: Kind of campaign selected for the equipment.

  • End Date: Date when this campaign will be deactivated.

  • Description: A short description of the campaign.

  • Status: Active or Inactive status of the campaign.

Warranty Coverages

You can view a list of all parts and machine policies associated with this equipment. The policies listed here include all standard, extended and goodwill policies registered on this equipment. Some of these policies may be active and others may be deactivated/expired.

 

The policy details are displayed in a tabular format, with the following information being shown for each policy:

  • Policy Code: Code assigned to the policy. This code uniquely determines a warranty policy. Clicking on this link will take you to the policy details page for this policy. For further information on the policy details page, click here.

  • Policy Name: Name of the policy.

  • Type: Kind of policy. A policy can be of type standard, extended or goodwill.

  • Start Date: Date when this policy will start providing warranty coverage for this equipment.

  • End Date: Date when the warranty coverage on this equipment, provided by this policy, will expire.

  • Months Covered: Number of months of warranty coverage provided by this policy.

  • Hours Covered: Number of hours of warranty coverage provided by this policy.

  • Status: Current status of the warranty coverage.

  • Comments: Comments mentioned while registering the policy.
