SHIPMENT TAG

A shipment tag is generated when you click on the Generate Shipment button in the Due Parts and the Overdue Parts folders. The number of shipment tags generated is equal to the number of unique return locations. The shipment tag also includes separate tags for each part.

 

The following details are shown in the shipment tag:

  • Bar Code: A bar code which uniquely identifies the shipment. This can be read by a scanning device. The shipment number is shown below the bar code. For further information, click here.

  • Ship To Address: The address to which the shipment is being done. The complete postal address of the location where you have been asked to return the part(s).

  • Shipped From: This section shows your name, your complete postal address and your dealer number.

  • Part Return Details: The details of the parts that are part of the shipment are displayed in a tabular format, showing the following details:

  • Claim Number: The number of the claim to which this part belongs.

  • Serial Number: Serial number of the machine or the part, depending on the claim type, on which you filed the claim.

  • Item Number: Item number of the machine or the part on which you filed the claim.

  • Model: Model of the machine or the part on which you filed the claim.

  • Part Serial Number: Serial number of the part being returned. If the part is non-serialized, then this field will appear blank.

  • Part Number: Part number of the part that is being returned.

  • Quantity: Number of parts of the given specification that are being returned as part of this shipment. This field will show 1 if the part is serialized.

  • Description: A short description of the part that is being returned.

You need to cut this shipment tag and paste it on your shipment package before shipping it.

 

A part tag is generated for each part included in the shipment. You can see the following details for each part in the part tag:

  • Bar Code: A bar code which uniquely identifies the part. This can be read by a scanning device. The part description is shown below the bar code.

  • Claim Number: The number of the claim to which this part belongs.

  • Serial Number: Serial number of the machine or the part, depending on the claim type, on which you filed the claim.

  • Part Number: Part number of the part that is being returned.

You need to cut each part tag and paste it on the package of that part before placing it in the shipment.
