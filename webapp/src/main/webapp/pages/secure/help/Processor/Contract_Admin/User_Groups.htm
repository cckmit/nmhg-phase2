User groups

The Warranty Management application allows you to group various users and arrange the groups in a hierarchical tree structure known as schema.

 

To view user groups page, click on the Contract Admin tab on your left-hand pane. All folders under this tab are shown. Click on the User Groups link under the Manage Groups folder. You can view all currently available schemas listed in a tabular format. To know more about navigating through the table, see Lists.

 

You can either create a new schema or add/remove groups from existing schemas.

 

Creating Schemas

To create user groups, it is first necessary to create a schema. You can then create various user groups, at different levels in this schema.

 

To create a schema,

  • Click on the Create Schema button. The New Item Schema page is shown.

  • In the Name field, enter a name with which you wish to save this schema.

  • In the Purpose selection box, you can view all purposes that are currently not attached to any schema. Select a purpose for your schema.

  • Click Cancel to exit from this page without saving any details entered by you.

  • Click Save to save this schema. The schema is saved and the Manage Member Groups button is shown on the New Dealer Schema page.

Note: You can attach any number of purposes to a schema but a purpose can be attached to one schema only.

 

Creating User Groups

You can create user groups at various levels inside a schema.

 

To create a user group,

  • Click on the Manage Member Groups button on the New User Schema page. The Manage User Groups for <schema name> page is shown.

  • Click on the Create User Group button. The New User Group page is shown.

  • In the Group Name field, enter a name with which you wish to save this group.

  • In the Group Description field, enter a short description for the group.

  • Click Cancel to discard the new group and exit this page.

  • To create a group consisting of other user groups, select the Consisting of Groups radio button.

  • To create a group consisting of users, select the Consisting of Users radio button.

  • Click Add Users/Add Groups button. A pop-up window is shown. Enter the following details in this window:

  • In the User Name/Group Name field, enter the names of users or the group name of the groups you wish to add to your group.

  • In the Description field, enter the description of the user groups which you wish to add to your group.

  • Click Search. All users/groups with the given parameters will be listed.

  • Select the check-box against the user/groups that you wish to add to your group.

  • Click Add Users/Add Groups button. The selected users/groups are added to your group.

  • Click on Create User Group button. The group with the selected users/groups is created.

You can add new users or user dealer groups to existing groups or remove users or user groups from them.
