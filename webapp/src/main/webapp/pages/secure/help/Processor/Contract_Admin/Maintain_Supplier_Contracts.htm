MAINTAIN SUPPLIER CONTRACTS

A supplier contract consists of all the terms and conditions agreed to between the supplier and the manufacturer. It consists of rules as to when a particular part is recoverable and under what conditions. It further contains information regarding the expenses that are covered by the supplier.

 

To view all contracts,

  • Click on the Maintain Supplier Contracts link on your Home page under the Common Actions Folder.

  • Click on Maintain Supplier Contracts folder under the Contract Admin tab on your left-hand pane.

Either of the above actions take you to the Maintain Supplier Contracts page. On this page you can see the Supplier Contracts table that lists all existing supplier contracts. For further information on navigating through the table, see Lists.

 

The Supplier Contracts table provides you with the following details about each contract:

  • Contract Code: A unique code assigned to the contract.

  • Contract Name: Name of the contract.

  • Supplier: Name of the supplier with whom this contract is agreed to.

  • Validity From: Date from which the contract covers the mentioned parts.

  • Validity To: Date from which the contract ceases to cover the mentioned parts.

If you double click on any entry in this table, all details relevant to that contract will be displayed on a new page, Contract Code <n>. The detail page is divided into the following sections, Contract Details, Compensation Terms and Items Covered for ease of navigation.

 

Creating a Contract

To create a contract,

  • Click on the button.

  • The New Contract page is shown.

This page is divided into various sections. You need to enter appropriate details in each of these sections to create and save a valid contract.

 

Contract Details

You can view the following details about the contract in this section:

  • Contract Code: Enter a unique code for the contract.

  • Contract Name: Enter a name for the contract, as agreed between the supplier and you.

  • Supplier: Use this drop-down field to select the name of the supplier with whom this contract is associated.

  • Shipment Location: Use this drop-down field to select a supplier location to which a supplier sourced part must be shipped in case of failure. The options in this drop-down field depend on the supplier you select in the Supplier field.

  • Validity From: Use the option to select a date from which this contract will start covering the supplier's parts.

  • Validity To: Use the option to select a date from which this contract ceases to cover the parts.

  • Coverage Duration: Enter the number of months for which the supplier parts are covered by this contract. This duration normally starts ticks off either when the part is installed on whole truck or is bought by an end consumer.

  • Hours on Truck: Number of hours the part was in service till its failure.

  • Is Physical Shipment Needed: This flag determines whether the supplier requires actual packaging and shipment of the part in order to agree to the recovery terms. Select this check-box if the supplier requires a physical shipment of the part. If this check-box is un-selected, then only the electronic claim is sent to the supplier.

  • Is Processor Review Needed?: This flag determines whether processor review is required on the part.

  • Is AutoDebt Enabled?: This flag determines whether the amount due for a part in any of the supplier's folders will be automatically debited from the supplier's account.

  • Carrier: Name of the carrier in which the consignment is shipped.

  • Shipment Account Number: The shipment number of the contract.

  • Recovery Based on: A recovery can be based on Causal part or Removed parts. Select either of the one.

Compensation Terms

This section captures all cost categories that will be covered by this contract. You can specify whether the parts will be replaced by the supplier or reimbursed for. You can also set up a maximum amount that will be reimbursed for the part and each cost category.

 

The following cost categories are listed here: OEM Parts, Non-OEM Parts, Miscellaneous Parts, Labor, Travel, Rental, Item Freight Charges, Meals, Parking, Tolls, Freight and duty. For each of these cost categories, except for the OEM Parts, you have the following two radio button options:

  • Not Covered: Select this option if this cost category will not be reimbursed by the supplier contract.

  • Covered: Select this option if this cost category will be reimbursed by the supplier contract. If you select this option, the following expression is displayed:

X % of part cost + $ Y subject to maximum of $ Z, where,

X is a non-negative value, that is, any number above and including zero

Y is any number, positive or negative

Z is any non-negative number

Enter the numerals for X, Y and Z as agreed between the supplier and you. You can claim a maximum of Z amount for this cost category.

 

For the OEM Parts cost category, the following radio button options are shown:

  • Installable: Select this option if the faulty part will be replaced by the supplier. If you select this option, then you will not be paid for the OEM Parts cost category in the claim.

  • Reimbursable: Select this option if the faulty part can be replaced by any other part of similar specifications and the cost of the new part will be reimbursed by the supplier. If you select this option the expression shown above will be displayed.

Items Covered

In this section, you can choose the items that are covered by this contract. You can add any number of parts in this section, as agreed between the supplier and you. To add a part, click on the option. A new row is shown under this section. Enter the following details in this row:

  • Part Number: use this drop-down field to select the part number of the supplier's part that will be covered by this contract. All parts sourced from the selected supplier will be listed in this drop-down field.

  • Supplier Part Number: Enter the OEM equivalent part number for the selected supplier part. This field supports auto-complete.

  • Supplier Part Description: A short description of the selected part is shown in this field.

To delete a row, click on the option corresponding to that row.

 

Actions

You can perform the following actions on this page:

  • Cancel: Click on this button if you want to close this page without saving any details entered by you.

  • Submit: Click on this button to save the contract. The contract will come into force on the specified contract start date.

The contract will be listed in the Supplier Contracts table. You can edit any contract in this table by opening it in the detailed page. You cannot edit the contract code, supplier names in an already created contract.

 

To deactivate a contract, change the Valid To date to the current date. The contract will be deactivated from the next day.
