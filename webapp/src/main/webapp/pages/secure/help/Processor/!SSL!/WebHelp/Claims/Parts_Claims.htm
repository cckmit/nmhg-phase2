PARTS CLAIM

You can submit a parts claim when the equipment is out of warranty but the failed part is still covered by individual part warranty.

 

To create and submit a part claim:

When you click on the Create Claim option, a new page, New Claim is shown. You are now on Page 1 of 2 on this tab. You need to provide the following details on this page to submit a parts claim:

  • Dealer: Enter the name of the dealer on whose behalf you are submitting this claim.

  • Use the Claim Type drop-down option to select the Parts Claim option.

A parts claim can be serialized or non-serialized. If you have the serial number of the item, you can file a serialized claim. A non-serialized machine claim is filed when the serial number of the equipment is not available on which you are submitting the claim or if the system fails to recognize the serial number.

  • Have part serial number:

  • If it is a serialized part claim, select ‘Yes’ and choose any one of the following part installed selection.

Serialized part with Part installed on serialized host machine.

Serialized part with Part installed on non-serialized host machine.

Serialized Part with Part installed on Competitor Model.

  • If it is a non-serialized part claim, select ‘No’ and choose any one of the following part installed selection.

Non-serialized part with Part installed on serialized host machine.

Non-serialized part with Part installed on non-serialized host machine.

Non-serialized part with Part installed on competitor model.

 

Few of the fields in the New Claim page vary depending upon the selection of claim type that can be serialized or non-serialized part and the part installed selections. To view instructions for submitting the above types of part claim, click on the links above.
