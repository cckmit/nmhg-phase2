EMAIL SUBSCRIPTION

The IR Warranty System has the feature to send Email Alerts to the users. Currently this system has different users like Admin, Dealer, Processor, SRA and Supplier. The system displays different email subscription events based on the user roles. If the user does not want all email alerts, then the user has the choice to select the events from the list of Email Subscription Events which are displayed for their roles.

 

To view and select email subscription events, click on Email Subscription tab under the Settings tab on the left-hand pane. The Processor contains the following events under Email Subscription Events Section:

  • Claim Transferred

  • Mismatch of Part on/Part off coverage

  • Claim Processor Review

  • Claim Replied Back

  • Pending Authorization

  • Parts Rejected On Claim

  • Rejected Part Returns

  • Waiting For Labor FOC Claims

  • Part Received

  • Claim Appealed

Once the user selects the event for which email notification is required, click Update.
