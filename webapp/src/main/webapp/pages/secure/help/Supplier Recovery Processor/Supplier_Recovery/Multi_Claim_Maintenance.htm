MULTI CLAIM MAINTENANCE

The Warranty Management application allows you to add attributes to the claim submitted by any service provider.

 

To add claim attributes,

  • Click on the Multi Claim Maintenance folder under the Supplier Recovery tab on the left-hand pane. You can search for claims based on various claim parameters on this page.

You can search for positive results for a set of conditions that you define on this page, or you can also search for a negation of the set of conditions defined on this page.

 

 

The conditions created under one search phrase can be applied in different ways.

  • Keep the Not check-box un-checked. Select the option All of the conditions are true in the drop-down. The search results will show all claims which satisfy all of your search conditions.

  • Check the Not check-box. Select the option All of the conditions are true in the drop-down. The search results will show all claims which do not satisfy any one of your search conditions.

  • Keep the Not check-box un-checked. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all claims which satisfy at least one of your search conditions.

  • Check the Not check-box. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all claims for which do not satisfy any your search conditions.

Every search condition is made up of three parts, <attribute> <object> <value>.

The attribute is a property of the business object, i.e. the warranty claim. An operator is a mathematical operation used as part of the expression. The value part of the search condition indicates the operand against which the operator should be applied to the attribute. You will need to create a search condition using the attributes, objects and values provided in the query builder to search for claims.

 

Enter the following details to search for claims:

  • If you want to view the results for a negation of the search conditions you create, then check the NOT check-box.

  • Select an option from the drop-down, All of the conditions are true or Atleast one of the conditions below is true.

  • In the Create Business Condition section, click on the option. A new row will be added.

  • Use the drop-down to select an attribute for your search condition. For example, select Caused By from this drop-down.

  • Use the drop-down to select an object for your condition. The options in this drop-down depends on the attribute selected by you. For example, for the Caused By attribute, select contains from this drop-down.

  • Now you need to enter a value in the text field. This is a free text field. For example, enter the value Bad Crimp in this field.

  • You can add multiple conditions for a more advanced search, by clicking on the option.

  • You can delete a condition by clicking on the corresponding to that row.

  • If you want to search for claims based on the condition you just created, click on the Execute button.

The page refreshes to show the search results in a tabular format.

  • Select the checkbox for the claims for which you want the attribute to be associated.

  • Under the Attribute section, select the attributes. There are different types of attributes: Acceptance Reason, Dispute Reason, RGA Number, Carrier and Location. You can select one or more attributes.

  • Click Submit.

Depending upon your selection, a particular text box will appear when the claim is in the processor inbox. The processor can enter the comments in that text.
