BEFORE YOU BEGIN

This chapter includes the basic information needed to login to the Warranty Management application. It also provides details on the User Interface of the application and its Home page.

 

Logging In

To access the Warranty Management application, you must have a valid user account. If you have any clarifications regarding your user account, consult the Warranty Administrator.

 

To log in to the Warranty Management application:

  • Open the application in your Web browser. The Login page appears.

  • In the User ID field, enter your user account name. All user accounts are set up by the Warranty Administrator.

  • In the Password field, enter the password associated with the user account.

  • Click Login. The Home page appears.

User Interface

The user interface of the Warranty Management application is designed to be simple and easy to use. Each page gives you information about the Current User, Current Location, and the Current Folder.

  • Left Hand Pane

On the left hand pane you can see various tabs. You can navigate to any of these tabs by clicking on the tab name.

 

When you click on a tab name, the selected tab expands and the folders under that tab are shown. You can navigate to any folder in a tab by clicking on the folder name. A page corresponding to the selected folder will appear on the right hand pane.

  • Right Hand Pane

On the right hand pane you will see the current folder page. Observe a tabular list displaying all items of the selected folder on this page. Refer Lists for further information.

 

The right hand pane also displays the following actions: Download to Excel, Print, Refresh and Delete.

 

Home

By default, the Warranty Management application opens on this page. The Home page shows a list of various folders and provides you with short-cuts to various actions supported by the application.

 

The top right corner of the Home page displays the message Welcome <username>. The current date is displayed in the top left corner, in the following format: <day>, <month> <dd>, <yyyy>.

For example: Friday, March 08, 2013.
