CLAIMS UNDER RECOVERY

All claims under recovery will be listed under this section under four different types of inboxes. Clicking on any inbox displays all the claims available under that particular inbox in the selected view. This page also displays two dropdowns for creation of custom views:

Actions: This dropdown enables two actions- Creation of a new view and edit selected view.

To create a new view, click Create View from the Actions dropdown. The Create Inbox View page is displayed. It displays the following fields:

  • View Name: Enter the name of the view.

  • Available Fields: All the fields available or creating a view are displayed in this section such as Acceptance Reason, Amount Asked, Amount Credited, Assign To etc. You can select a maximum of 12 fields together for creating a view.

  • Selected Fields: All the fields selected by you are displayed in this section.

  • Default Sorting Criteria: Select the default sorting criteria from this dropdown.

  • Sorting Order: Select the sorting order as either ascending or descending.

After entering all the details, click Submit to create this view.

 

View: This dropdown displays all the views available. Select the desired view from the dropdown. This can be either the already existing view or the view which you have created. The table displayed in this page displays the fields present in the selected view. By default, the default view is displayed.

 

You can see the following details for each claim in the default view:

 

The table provides you with the following details about each claim:

  • Recovery Claim Number: The recovery claim number of the claim.

  • Supplier: The name of the supplier for this claim.

  • Supplier Number: The Supplier Number of the supplier.

  • Recovery Amount: The recovery amount for that claim.

  • Return Part status: The return part status of the claim.

  • Location: The location of the claim.

  • Status: The current status of the claim.

Double click on any entry in this table to display that specific claim details. The detail page is divided into the following sections, Claim Details, Truck Details, Failure Information, Service Information, Components Replaced, Comments, Manage documents, Recovery Cost Details, Recovery Claim History, Components Recovered, Part Return History, Recovery Comments and Decision for ease of navigation.

For further information on Claim Details page, click here.

 

The various inboxes present under Claims Under Recovery are explained as follows:

  • Disputed: A warranty claim comes in this inbox when the SRA or processor dispute a claim.

  • Accepted: If the processor accepts a claim, then the claim comes into this inbox. Clicking on a claim present in this inbox displays all the claim details which are not editable.

  • Awaiting Supplier Response: When a SRA or a processor transfers a claim to a supplier, then the claim remains in this inbox till the supplier to which the claim has been transferred takes any decision on that claim. Clicking on a claim present in this inbox displays all the claim details which are not editable.

  • Not for Recovery: If a supplier recovery processor takes the decision as 'Not for Recovery' on a recovery claim present in the New inbox, then the claim comes into the Not for Recovery inbox of the warranty processor. Here the warranty processor can accept or reject the claim. Clicking on a claim present in this inbox displays all the claim details which are not editable.
