CLAIMS DUE FOR RECOVERY

All claims due for recovery will be listed under this section under five different types of inboxes. Clicking on any inbox displays all the claims available under that particular inbox in the selected view. This page also displays two dropdowns for creation of custom views:

Actions: This dropdown enables two actions- Creation of a new view and edit selected view.

To create a new view, click Create View from the Actions dropdown. The Create Inbox View page is displayed. It displays the following fields:

  • View Name: Enter the name of the view.

  • Available Fields: All the fields available or creating a view are displayed in this section such as Acceptance Reason, Amount Asked, Amount Credited, Assign To etc. You can select a maximum of 12 fields together for creating a view.

  • Selected Fields: All the fields selected by you are displayed in this section.

  • Default Sorting Criteria: Select the default sorting criteria from this dropdown.

  • Sorting Order: Select the sorting order as either ascending or descending.

After entering all the details, click Submit to create this view.

 

View: This dropdown displays all the views available. Select the desired view from the dropdown. This can be either the already existing view or the view which you have created. The table displayed in this page displays the fields present in the selected view. By default, the default view is displayed.

 

You can see the following details for each claim in the default view:

  • Recovery Claim Number: The recovery claim number of the claim.

  • Supplier: The name of the supplier for this claim.

  • Supplier Number: The Supplier Number of the supplier.

  • Recovery Amount: The recovery amount for that claim.

  • Created Date: The date on which the claim was created.

  • Return Part status: The return part status of the claim.

  • Status: The current status of the claim.

Double click on any entry in this table to display that specific claim details. The detail page is divided into the following sections, Claim Details, Truck Details, Failure Information, Service Information, Components Replaced, Comments, Manage documents, Recovery Cost Details, Recovery Claim History, Components Recovered, Part Return History, Recovery Comments and Decision for ease of navigation.

 

The various inboxes present under Claims Due for Recovery are explained as follows:

  • New: A warranty claim comes in this inbox when it becomes a supplier recovery claim. Any recovery claim sent to the supplier by the supplier recovery processor (SRP) for his action is also added to this supplier inbox.

Note: In the default view, along with other claim details, the Return Part Status details are also displayed.

The supplier can choose to make any one of the following decisions on a claim on the New inbox.

  • On Hold

  • Cannot Recover

  • Transfer

  • Send to Supplier

  • Not for Recovery

  • Accept

  • In Progress: A supplier recovery claim comes in this inbox when the supplier selects the decision On Hold for the claim. The supplier can make the same decisions on this claim which he can make on a claim present in the New inbox.

  • Not for Recovery: If the SRP takes the action as 'Not for Recovery' on a recovery claim, the claim will move to Not for Recovery Request inbox of SRP and will stay in this inbox until the response is received from the warranty processor. At the same time the claim will move to the Not for Recovery inbox of the warranty processor where he can accept or reject the claim.

  • If the warranty processor rejects the claim, the claim will move from Not for Recovery Request inbox to Not for Recovery Response inbox of the SRP for further decision.

  • If the warranty processor accepts the claim, the recovery claim will be closed and will be removed from the Not for Recovery Request inbox.

  • Transferred: A supplier recovery claim comes in this inbox when the supplier selects the decision Transfer for the claim, and selects a person to whom he wants to transfer the claim from the drop-down present. The supplier to which this claim is transferred can make the same decisions on this claim which he can make on a claim present in the New inbox.
