VIEW PART SOURCE HISTORY

The View Part Source History page lists all the parts along with their description. It also displays any supplier associated with the part.

 

To see the list of parts, click on View Part Source History link under the Contract Admin tab on the left pane. The right pane displays the item numbers of the part along with the description. See Lists for more information on navigating through this table.

 

If you double click on any entry in this table, you can see the supplier details associated will this item. You can view the following details in the Item Number <n> page:

  • Supplier: Name of the supplier.

  • Supplier Part Number: Part number of the part supplied by the supplier.

  • Supplier Part Description: A short description of the part supplied by the supplier.

  • From and To Date: Date range during which the supplier covers the part.
