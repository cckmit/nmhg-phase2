PROFILE

The user with login access to the application can maintain personal details on the Profile page. The user can add or edit the details on this page.

 

Click on the Profile tab under the Settings tab on the left-hand pane to add/edit the following details,

  • Login Name: User name used to log into the application.

  • Locale: Language selected by the user.

  • Address, Country, State, City: Complete postal address of the user along with city, state and country details.

  • Postal/Zip Code: Postal code of the address location.

  • Phone: Contact number of the user.

  • Email: Email address of the user.

  • Fax: Fax number of the user.

If you want to close this page without saving any details entered by you, click Cancel.

Click Update to save the details entered by you.
