EMAIL SUBSCRIPTION

The IR Warranty System has the feature to send email alerts to the users. The system displays different email subscription events based on the user roles. If the user does not want all email alerts, then the user has the choice to select the events from the list of Email Subscription Events which are displayed for their roles.

 

To view and select email subscription events, click on Email Subscription tab under the Settings tab on the left-hand pane. The Advisor contains the following events under Email Subscription Events Section:

  • Claim Sent For Advice Request

Once the user selects the event for which email notification is required, click Update.
