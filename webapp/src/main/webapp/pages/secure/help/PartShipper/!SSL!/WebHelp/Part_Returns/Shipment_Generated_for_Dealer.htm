Shipment generated for dealer

When you generate a shipment for a part or a group of parts for the dealer, the details of the shipment are shown in the Shipment Generated for Dealer folder. You can update the shipping details for each shipment here. You can also add/remove parts from a previously decided shipment, as per the dealer's request.

 

To view the shipment details, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the Shipment Generated for Dealer folder. On this page you can see the Shipment Generated table that lists all generated shipments.

The Shipment Generated table provides the following information about each consignment:

  • Shipment Number: Identification number for the consignment. This number is generated when you click Generate Shipment.

  • Return Location: Name of the Return location to which you are making the shipment.

Double clicking on a shipment will open it in a new page. The detail page is divided into three sections, Claim Details, Part Return Details, and Shipment Information for ease of navigation. The Claim Details section displays information about the claim to which the part belongs. The Part Return Details will show all the parts that are to be returned to this location from the same claim. The Shipment Information section will show information about the shipment.

 

You can see the following details in the Claim Details section, in the detail page:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Truck Serial Number: This field displays the serial number of the machine. The Serial Number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: This field will show the model number of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed in the detail page:

  • Select Checkbox: If you want to include this part in your shipment, then select this check-box.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • WPRA: WPRA number of the shipment.

  • Return directly to supplier: This field displays Yes if this part needs to be returned directly to the supplier.

  • Pending Shipment: The status of the shipment.

  • Shipment Generated: The quantity of the parts for which shipment is generated.

  • Cannot Be shipped: The quantity of the parts which cannot be shipped.

  • Shipped: the quantity of the parts shipped.

  • Total:  The total number of parts.

In the Shipment Comments section, the shipment comments are displayed:

Shipment Information

This section displays the following details:

  • Shipment Number: The shipment number of this shipment.

  • Available Date: Select the available date and time of the shipment.

  • Load Type: This field allows you to select the Load Type from the dropdown.

  • Load Dimension: Enter the shipment load dimension data in this field.

  • Comments: If you have any additional comments about the shipment, parts being shipped, carrier used for the shipment, then you can enter the same in this field.

Actions

You can perform the following actions on the shipment in this folder:

  • Print Shipment Tag: By clicking this button, you can print the Shipment Tag.

  • Submit: If you want to submit this shipment, then click this button. This indicates that you have shipped the parts included in this shipment to the specified location. A success message will be shown confirming your action.
