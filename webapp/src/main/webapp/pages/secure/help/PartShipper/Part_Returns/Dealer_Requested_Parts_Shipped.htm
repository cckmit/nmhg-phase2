DEALER REQUESTED PARTS SHIPPED

When the Dealer submits a claim and ships the parts to the OEM, the part inspector either accepts or rejects the parts. If the Part Inspector rejects the parts, and the processor denies the claim, the claim  moves to the dealer's Rejected Parts inbox.  Once the dealer has submitted the claim and requested for the parts, it moves to the part shipper’s Dealer Requested Part inbox. When the Part Shipper generates Shipment, the Part is shipped and is seen in the Part Shipper's Dealer Requested Parts Shipped inbox.

To view the list of parts in this inbox,

  • Click on the Dealer Requested Parts Shipped link under the Action Folders column on your Home page.

  • Click on the Part Returns tab on the left-hand pane. The tab opens to display the folders under it. Click on the Dealer Requested Parts Shipped folder.

Either of the above actions take you to the Dealer Requested Parts Shipped page. This page displays the following shipped parts details.

  • Shipment Number: This field displays the Shipment Number of your shipment.

  • Return Location: This field displays the return location of your shipment.

  • Carrier: This field displays the shipper through which the parts are being shipped.

  • Tracking Number: This field displays the unique tracking number provided by the shipper for your shipment.

  • Shipment Date: This field displays the date and time of shipment.

Double clicking on an entry in this table displays the Details page. The detail page is divided into the following sections, Claim Details, Part Return Details,and Shipment Informationx:

 

The following details are displayed in the Claim Details section::

  • Claim Number: The number of the claim to which the part to be returned belongs. The Claim Number is a hyperlink to the Claim Detail page. Click here for more information on the Claim Detail  page.

  • Truck Serial Number: This field displays the serial number of the machine. The Serial Number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: This field will show the model of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed:

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • RGA Number: RGA number of the part which is shipped.

  • WPRA: WPRA number of the part which is shipped.

  • Return Directly to Supplier: Number of parts of the given specification that need to be shipped directly to the supplier.

  • Pending Shipment: Number of parts of the given specification for which shipment is pending.

  • Shipment Generated: Number of parts of the given specification for which shipment is generated.

  • Cannot Be Shipped: Number of parts of the given specification that cannot be shipped.

  • Shipped: Number of parts of the given specification that are shipped.

  • Total: The total number of parts shipped.

To print the shipment tag, click Print Shipment Tag.
