DEALER REQUESTED PART

When the Dealer submits a claim and ships the parts to the OEM, the part inspector either accepts or rejects the parts. If the Part Inspector rejects the parts, and the processor denies the claim, the claim  moves to the dealer's Rejected Parts inbox.  Once the dealer has submitted the claim and requested for the parts, it moves to the part shipper’s Dealer Requested Part inbox.

To view the returnable part list,

  • Click on the Dealer Requested Part link under the Action Folders column on your Home page.

  • Click on the Part Returns tab on the left-hand pane. The tab opens to display the folders under it. Click on the Dealer Requested Part folder.

Views

Two types of views are available for you to view the data in the Dealer Requested Part list page: Location View and Claim View.

 

Location View

The location view consolidates all the parts across the claims that are to be shipped to the same location. To view the parts in this view, click the Location View button. You can see all parts grouped by return locations,  in a tabular format.

 

The Location View table provides the following details about each return location:

  • Dealer Name: The name of the dealer for this part.

  • Dealer Number: Dealer Number for this part.

Double clicking on an entry in this table displays the Details page. The detail page is divided into the following sections, Claim Details, Part Return Details, Comments From Dealer and Comments:

 

The following details are displayed in the Claim Details section::

  • Claim Number: The number of the claim to which the part to be returned belongs. The Claim Number is a hyperlink to the Claim Detail page. Click here for more information on the Claim Detail  page.

  • Truck Serial Number: This field displays the serial number of the machine. The Serial Number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: This field will show the model of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed:

  • Select Checkbox: If you want to include this part in your shipment, then select this checkbox.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • Dealer Requested Parts: The quantity of the parts requested by the dealer.

  • Ship: Number of parts of the given specification that are shipped.

  • Cannot Ship: Number of parts of the given specification that cannot be shipped.

  • Dealer Name: The name of the dealer for this part return.

Claim View

The claim view consolidates all parts belonging to one claim, that must be returned to the OEM. The parts may have different return locations. To view the parts in this view, click on the Claim View button. You can see all parts belonging to individual claims, grouped together in a tabular format.

 

The Claim View table provides the following details about each claim:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

Double clicking on a claim will open it in a new page, <claim number>. The detail page is divided into the following sections: Claim Details, Part Return Details, Comments From Dealer and Comments.  These sections are discussed in the Location View above.  

Actions

You can generate the shipment after providing the required details and clicking on Generate Shipment button.
