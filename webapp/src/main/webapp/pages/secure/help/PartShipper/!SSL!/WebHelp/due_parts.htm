Due Parts

Once the WPRA (Warranty Part Return Authorization) is generated, the dealer is notified about the entire list of returnable part details in his Due Parts inbox.

To view the parts in the Due Parts inbox, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the Due Parts folder. On this page you can see the Due Parts table that lists all parts due to be returned to the OEM.

Note: You may need to collect relevant information from the dealer, about parts to be included in a shipment, before generating the shipment tags. Since the dealer has no access to the Warranty Management system, the information exchange will have to be done off-line.

Views

Various views are available for you to view the data in the Due Parts table. Use the View drop-down to select a view. The data will be re-arranged to reflect the view chosen by you. The following views are available in this drop-down, Location View, Claim View and Default Parts View.

 

Location View

The location view consolidates all the parts across the claims that are to be shipped to the same location. To view the parts in this view, select the Location View option in the View drop-down. You can see all parts grouped by return locations,  in a tabular format.

 

The Location View table provides the following details about each return location:

  • Location Name: Name of the return location.

  • Location Address: Complete postal address of the return location.

You can view the details of any return location by clicking on it. Double clicking on a return location will open it in a new page, <location name>. The detail page is divided into two sections, Claim Details and Part Return Details, for ease of navigation. The Claim Details section displays information about the claim to which the part belongs. The Part Return Details will show all the parts that are to be returned to this location from the same claim.

 

In the Claim Details section you can view the following details for the selected location, in the preview pane and the detail page:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a field product improvement claim, this field will show the serial number of the machine on which you filed the field product improvement claim.

  • Part Serial Number: If the claim was a parts claim, then you can see the part serial number of the part on which the claim was filed. If the part is non-serialized, then this field will appear blank.

  • Item Number: If the claim was a machine claim, this field will show the item number of the machine.

  • Part Number: If the claim was a parts claim, this field will show the part number of the part on which the claim was filed.

  • Model: This field will show the model of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed in the preview pane and the detail page:

  • Select Checkbox: If you want to include this part in your shipment, then check this check-box.

  • Serial Number: Serial number of the part that is to be returned. If the part is non-serialized, then this field will appear blank.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • Quantity: Number of parts of the given specification that are to be returned. If the part is serialized this value will always be 1.

  • Due Date: Date on which the part is due for return. If you do not ship this part to the specified location by this date, this part will move to your Overdue Parts folder. Click here to know more about the Overdue Parts folder.

  • Due Days: Number of days between the current date and the due date for this part.

  • Comments: The latest external comments on this part.

Claim View

The claim view consolidates all parts belonging to one claim, that must be returned to the OEM. The parts may have different return locations. To view the parts in this view, select the Claim View option in the View drop-down. You can see all parts belonging to individual claims, grouped together in a tabular format.

 

The Claim View table provides the following details about each claim:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a campaign claim, this field will show the serial number of the machine on which you filed the campaign claim.

  • Part Serial Number: If the claim was a parts claim, then you can see the part serial number of the part on which the claim was filed, in this field. If the part is non-serialized, then this field will appear blank.

  • Item Number: If the claim was a machine claim, this field will show the item number of the machine.

  • Part Number: If the claim was a parts claim, this field will show the part number of the part on which the claim was filed.

  • Model: This field will show the model of the machine on which the claim was filed.

You can view the details of any claim by clicking on it. Double clicking on a claim will open it in a new page, <claim number>. The detail page is divided into two sections, Location Details and Part Return Details, for ease of navigation. The Location Details section displays information about the location where the part must be returned. The Part Return Details will show all the parts that are to be returned to this location from the same claim.

 

In the Location Details section you can view the following details for the selected location, in the preview pane and the detail page:

  • Location Name: Name of the return location.

  • Location Address: Complete postal address of the return location.

In the Part Return Details section, the following details are shown, in both the preview pane and the detail page:

  • Select Checkbox: If you want to include this part in your shipment, then check this check-box.

  • Serial Number: Serial number of the part that is to be returned. If the part is non-serialized, then this filed will show a "-".

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • Quantity: Number of parts of the given specification that are to be returned. If the part is serialized this value will always be 1.

  • Due Date: Date on which the part is due for return. If you do not ship this part to the specified location by this date, this part will move to your Overdue Parts folder. Click here to know more about the Overdue Parts folder.

  • Due Days: Number of days between the current date and the due date for this part.

  • Comments: The latest external comments on this part.

Default Part View

In this view, you can see the parts to be returned in a part-by-part view where each of the parts in the folder is listed as a separate row. To view the parts in this view, select the Default Parts View option from the View drop-down. You can see details of all parts in a tabular format.

 

The Parts table provides the following details for each part:

  • Part Serial Number: Serial number of the part that is to be returned. If the part is non-serialized, then this field will appear blank.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned

  • Return Location: Name of the location to which this part must be returned.

  • Due Date: Date on which the part is due for return. If you do not ship this part to the specified location by this date, this part will move to your Overdue Parts folder. Click here to know more about the Overdue Parts folder.

  • Due Days: Number of days between the current date and the due date for this part.

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

You can view the details of any part by clicking on it. Double clicking on a part will open it in a part detail page. The detail page is divided into two sections, Part Details and History, for ease of navigation. The Part Details section displays information about the part. The History section will show information about all comments given by various users on this part.

 

You can see the following details for the selected part in the detail page:

  • Part Details: This section shows part serial number, part number, description, quantity, return location, due date, claim number, claim fault code, claim fault found and claim caused by.

  • History: This section lists all the updates done on the part in the form of an audit trail with the following columns:

  • Date and Time: The date and time of the action that was taken on the part.

  • User: Name of the user who took the action on the part.

  • Status: The status in which the part ended up in after this update. Please refer to the part return life-cycle for the various statuses the part can be in.

  • Comments: Any comments by the user who performed the update.

Actions

The following actions are allowed, for all views in the Due Parts folder:

  • Generate Shipment: To generate a shipment, first select the check-boxes against the parts which you want to include in this shipment. Then click on the Generate Shipment button.

  • The Shipment Tag is displayed in a pop-up window. For more details on the shipment tag, click here.

The part is now moved to the Shipment Generated folder. A success message will be shown confirming your action. The message will hide itself and the current page will be closed.

 

Incase of part return, the part has to be returned to the receiver within the due date defined. If failed to return within the due date, then the claim move to the On Hold For Part Return inbox. This inbox list all the claims having part marked for return. Double click on the claim number to view the claim details.
