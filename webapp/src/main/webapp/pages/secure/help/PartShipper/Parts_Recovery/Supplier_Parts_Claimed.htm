SUPPLIER PARTS CLAIMED

The Supplier Parts Claimed folder contains details of parts that must be shipped to the specified supplier location for recovery and reimbursement. The data in this folder allows you to efficiently manage your parts recovery.

 

To view the parts that need to be shipped to the supplier locations, click on the Parts Recovery tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the Supplier Parts Claimed folder. On this page you can see the Supplier Parts table that lists all parts due to be returned to the suppliers. For further information on navigating through the table, see Lists.

 

The Supplier Parts table provides the following information:

  • Recovery Claim Number: The identification number of the claim to which the part to be returned belongs.

  • Part Number: Part number of the part that is to be returned.

  • Barcode: A bar code which uniquely identifies the part.

  • Product: The product name of the truck to which this part belongs.

  • Model: Model number of the truck to which this part belongs.

  • Supplier: The name of the supplier.

  • Return Location: Name of the location to which you are making the shipment.

You can view the details of any claim by clicking on it. Double clicking on an entry will open a new page, Return Location <location name>. The page displays the claim details along with the supplier parts associated. Once you select the parts to be shipped to the supplier, click on Generate Shipment button. The part is now moved to the Shipment Generated folder. A success message will be shown confirming your action.
