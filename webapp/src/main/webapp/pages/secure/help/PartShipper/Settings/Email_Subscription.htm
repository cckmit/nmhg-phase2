EMAIL SUBSCRIPTION

The Warranty System has the feature to send email alerts to the users. The system displays different email subscription events based on the user roles. If the user does not want all email alerts, then the user has the choice to select the events from the list of Email Subscription Events which are displayed for their roles.

 

To view and select email subscription events, click on Email Subscription tab under the Settings tab on the left-hand pane. The PartShipper contains the following events under Email Subscription Events Section:

  •      Claim Transferred  

  •      Claim Accepted And Closed  

  •      Claim Processor Review  

  •      Service Manager Review  

  •      Mismatch of Part on/Part off coverage  

  •      Claim Replied Back  

  •      Waiting For Part Return  

  •      Claim Forwarded  

  •      Claim Denied And Closed  

  •      Parts Rejected On Claim  

  •      Claim Denied For Lack Of Parts  

  •      Pending Authorization  

  •      Rejected Part Returns  

  •      Service Manager Response  

  •      Claim Sent For Advice Request  

  •      Claim Re-accepted  

  •      Part Received  

  •      Part Moved To Overdue  

  •      Waiting For Labor FOC Claims  

  •      Claim Appealed  

  •      Part To Be Returned  

Once the user selects the event for which email notification is required, click Update.
