DUE PARTS INSPECTION

The Due Parts Inspection folder contains details of all parts forwarded by the receiver for further inspection. The data in this folder allows you to efficiently manage all parts marked for inspection.

 

To view all parts marked for inspection, click on the Part Returns tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the Due Parts Inspection folder. On this page you can see the Parts for Inspection table that lists all parts received in this warehouse. For further information on navigating through the table, see Lists.

 

If you double click on any entry in this table, all details relevant to that part will be displayed on a new page, Claim Number <n>. You can view the following details in the Claim Number <n> page:

 

Claim Details:

  • Claim Number: Claim number of the claim associated with this part.

  • Part Serial Number: Serial number of the part.

  • Model Number: Model number of the part.

Part Return Details:

  • Part Serial Number: Serial number of the part.

  • Part Number: Part number of the part.

  • Description: A short description of the part.

  • Barcode: A unique identifier for the part.

  • Return Location: Location where the dealer is expected to ship this part.

  • Action: This drop-down menu allows you to perform various actions under the Accept and Reject decisions. If the claim is rejected, it will appear in the Rejected Part Return inbox.

Comments: If you have any additional comments on this part, you can write the same in this text field.

 

Click Submit to submit the actions taken by you on all parts marked for inspection. A success message will be shown confirming your action. The message will disappear and the current page will be closed. The part will be removed from your Due Parts Inspection folder.
