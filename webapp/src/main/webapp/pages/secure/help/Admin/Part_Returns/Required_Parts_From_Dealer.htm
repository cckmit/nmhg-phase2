Required Parts From Dealer

Parts configured for return will be marked for return only after the claim is accepted by the processor or manually requested by processor during processing.

If PRC is configured for the parts then the parts will be automatically flagged for return. If no PRC configuration is available for the parts, then processor can manually ask for return and thus specifies the return location and dealer’s pick up location.

After processor takes decision on the claim and if the claim is not rejected, then parts are added to Processor’s Required Parts From Dealer inbox.

To view the parts in the Required Parts From Dealer inbox, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the Required Parts From Dealer folder.

Views

Two types of views are available for you to view the data in the Required Parts From Dealer table: Location View and Claim View.

 

Location View

The location view consolidates all the parts across the claims that are to be shipped to the same location. To view the parts in this view, click the Location View button. You can see all parts grouped by return locations,  in a tabular format.

 

The Location View table provides the following details about each return location:

  • Dealer Name: Name of the Dealer.

  • Dealer Number: Number of the Dealer.

Double clicking on an entry in this table the detail page. This page is divided into two sections, Claim Details and Part Return Details: The Claim Details section displays information about the claim to which the part belongs. The Part Return Details displays information about all the parts that are to be returned to this location for the same claim.

 

The following details are displayed in the Claim Details section::

  • Claim Number: The number of the claim to which the part to be returned belongs. The Claim Number is a hyperlink to the Claim Detail page. Click here for more information on the Claim Detail  page.

  • Serial Number: This field displays the serial number of the machine. The Serial Number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: This field will show the model of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed:

  • Select Checkbox: If you want to include this part in your shipment, then select this checkbox.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • To be Shipped: Number of parts of the given specification that are to be returned.

  • Return directly to Supplier: Displays Yes if this part needs to be returned directly to the supplier.

  • Return Location: Displays the Return Location of the part.

  • RMA Number: The RMA Number of the part to be returned.

  • Dealer Name: The name of the dealer for this part return.

Claim View

The claim view consolidates all parts belonging to one claim, that must be returned to the OEM. The parts may have different return locations. To view the parts in this view, select the Claim View option in the View drop-down. You can see all parts belonging to individual claims, grouped together in a tabular format.

 

The Claim View table provides the following details about each claim:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

Double clicking on a claim will open it in a new page, <claim number>. The detail page is divided into two sections, Claim Details and Part Return Details. These sections are discussed in the Location View above.  

Comments

This page allows you to enter your comments in this section.

Actions

Two buttons are displayed at the bottom of this page: Generate WPRA and Remove Parts.

  • Generate WPRA: Click on this button to manually generate WPRA for the selected parts.

  • Remove Parts: Click on this button to remove the selected parts which you don't want to return back.

The parts for which WPRA is generated are thus moved from Required Parts From Dealer inbox to WPRA Generated for Parts inbox. Click here for more details on WPRA Generated for Parts inbox.
