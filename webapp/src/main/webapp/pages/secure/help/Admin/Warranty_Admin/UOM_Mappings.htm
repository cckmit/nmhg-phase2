UOM Mappings

UOM represents a “Unit of Measure” for a configured part. UOM actually refers to a scale which is used to measure the quantity of a specific item/object. Warranty administrator maps distinct UOM existing for business unit to a numeric value and to a new display value of UOM.

 

To create a UOM mapping,

  • Click on Maintain UOM Mappings tab under the Warranty Admin.

  • Click on the button in the listing toolbar and then click on the button to enter the following details:

  • Base UOM Value: Select the measurement unit from the drop-down.

  • New UOM Display Value: Enter the new display value for the measurement unit.

  • New UOM Mapping Value: Enter the mapping value for the new UOM display.

  • Click Submit.

Once the UOM is created, it will be listed on the Maintain UOM Mappings page.

 

To update the UOM mapping,

  • Click on the UOM entry listed on the Maintain UOM Mappings page.

  • Edit the mapping value for the new UOM.

  • Click Update.

A claim can be filed from any part of the world and thus Warranty Management application gives you the option to change the UOM display value as per the dealer's display or local language. The option to specify the internationalized values is available against each created UOM mapping. If you change the language of the dealer, the respective internationalized value is displayed for the mapped UOM.
