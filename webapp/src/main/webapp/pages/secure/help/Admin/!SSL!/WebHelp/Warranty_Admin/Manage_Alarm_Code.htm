MANAGE ALARM CODE

Alarm Codes are used by the dealers to give additional information about the product while filing a claim. To create an alarm code,

  • Click on Manage Alarm Code tab under the Warranty Admin tab on the left-hand pane or click on Manage Alarm Code under the Common Actions column on the Home page.

  • Click on the button on the listing toolbar.

  • Enter the following details under the Add Alarm Code section:

  • Code: Enter the name of the alarm code. This code will appear in the Alarm Code drop-down while filing a claim.

  • Description: Enter a short description for the alarm code.

  • Click on the button under the Add Product section to add a product to which you want to associate this alarm code. You can also add multiple products by clicking on the button.

  • Click Submit.

Now, when a dealer files a claim for the product to which alarm code has been associated, this particular alarm code will appear in the drop-down for selection.

 

Alarm Code is a business unit based flag set up by the admin for dealers. To enable a dealer to view alarm code section while filing a claim,

  • Click on Manage Business Configurations tab under the Warranty Admin tab on the left-hand pane.

  • Click on Claim Input Parameters under the Claim section in the Manage Business Configurations page.

  • Select the radio button ‘Yes’ for “Is alarm codes section visible.”

  • Click Update.

All the dealers can now view Alarm Code section while filing a claim.
