BATTERY TEST SHEET

For every part number (battery), there exist a battery test sheet which is created by the admin. On the warranty claim submission page 2, when dealer enters a replaced battery part for which a Battery test sheet is configured by admin, system automatically displays a button to enter the battery test sheet (BTS) and disables the Quantity field. As soon as dealer clicks on the button, system displays a pop-up to enter the Battery Test sheet against the serial number. In case of multiple inventories on a claim, the system will display a link against each serial number.

 

Perform the following steps to create a Battery Test Sheet:

  • Click on Manage Business Configurations tab under the Warranty Admin tab on the left-hand pane.

  • Click on Create Battery Test Sheet under Manage Battery Test Sheet to display the Create Battery Test Sheet page. This page contains 5 sections:

  • Create Battery Test sheet: Enter the battery details in this section such as Part Number and Battery Test Sheet Label. The Part Description field is automatically populated after you enter the Part Number.

  • Test 1: Full Charge Battery Test: Enter the details in the sections Labour Hours and Validation Parameters.

  • Test2: On Charge Voltage Test: Enter the details in the sections Labour Hours and Validation Parameters.

  • Test3: Hydrometer Test: Enter the details in the sections Labour Hours and Validation Parameters.

  • Test4: Discharge Test: Enter the details in the sections Labour Hours, Validation Parameters and Temperature.

  • Click Save to save the entered details. If you want to deactivate a particular test, select the Deactivate Test checkbox under the desired section and click De-Activate.

  • On clicking Save, this message is displayed: Battery Test Sheet Updated Successfully.
