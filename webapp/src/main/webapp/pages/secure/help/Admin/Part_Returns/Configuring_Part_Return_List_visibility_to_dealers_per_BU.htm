Part RETURN LIST VISIBILITY TO DEALERS

The Part Returns section is designed to help you manage the process of returning replaced parts that are requested back by the OEM. As an administrator, you can configure the parts return list visibility as per the Business Unit.

 

Perform the following steps to enable dealers to view the part return configuration list:

  • Click on Manage Business Configurations tab under the Warranty Admin tab on the left-hand pane. Click on Return Part Management section on the Manage Business Configurations page.

  • Select the radio button Yes for Enable Dealers to view Part Return list.

  • Click Update.

All the dealers will now be able to view the part return configuration list. For further information on Managing Business Configurations, click here.
