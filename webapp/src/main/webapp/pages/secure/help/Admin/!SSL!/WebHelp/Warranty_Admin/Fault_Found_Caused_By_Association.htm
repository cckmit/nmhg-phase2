Fault Found/Caused By Association

The Warranty Management application allows you to configure and view the "fault found-caused by" mapping for each product/model. Fault found represents the “Malfunction Code”, and Caused by helps to capture what caused the failure in a structured manner.

 

To view the Fault Found/Caused By Association page,

  • Click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Failure Structure folder. All links under this folder are shown. Click on the Fault Found/Caused By Association link.

This page enables you to create the mappings directly on the selected model or the product. This page also allows you to view the failure information for the selected model/product.

 

To create a Fault Found/Caused By Association:

  • Use the Product/Model/Item Group drop down option to select an option for which you want to view the mapping. This drop-down option lists all models of all trucks manufactured by the OEM.
