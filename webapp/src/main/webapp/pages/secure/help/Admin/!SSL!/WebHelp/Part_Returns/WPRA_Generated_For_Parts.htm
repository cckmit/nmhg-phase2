wpra generated for Parts

Parts configured for return will be marked for return only after the claim is accepted by the processor or manually requested by processor during processing.

If PRC is configured for the parts then the parts will be automatically flagged for return. If no PRC configuration is available for the parts, then processor can manually ask for return and thus specifies the return location and dealer’s pick up location.

After processor takes decision on the claim and if the claim is not rejected, then parts are added to processor’s ‘WPRA Generated for Parts inbox.

Once in a month the Warranty Part Return Authorization (WPRA) is generated for the entire returnable parts.

 

To view the parts in the WPRA Generated for Parts inbox, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the WPRA Generated for Parts folder. On this page, you can see the WPRA table that lists all parts for which WPRA is generated.

  • WPRA Number: This column specifies the WPRA Number of the  part.

  • WPRA ID: This column specifies the WPRA ID of the part.

  • Generate Date: This column specifies the date and time when WPRA was generated for that part.

  • Return Location: This column specifies the return location of this part.

  • Dealer Name: This column specifies the dealer name for this part.

Double clicking on an entry in the list page displays it in a new page. This page is divided into two sections, Claim Details and Part Return Details. The Claim Details section displays information about the claim to which the part belongs. The Part Return Details will show all the parts that are to be returned to this location from the same claim.

 

In the Claim Details section you can view the following details:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the Claim Detail page.

  • Serial Number: The serial number of the claim. The serial number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: The model number of the truck on which the claim was filed.

In the Part Return Details section, the following information is displayed:

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • To Be shipped: Number of parts of the given specification that are to be returned.

  • RMA Number: RMA number of the part that is to be returned.

  • WPRA Number: WPRA Number of the part that is to be returned.

  • Dealer Name: Dealer Name for the part that is to be returned.

Actions

If the processor decides not to return the part then he can remove the items from the WPRA inbox. Thus, although the detail page is read only, it allows the following action:

  • Cancel Return Process: To cancel the return process for a part, click the Cancel Return Process button.
