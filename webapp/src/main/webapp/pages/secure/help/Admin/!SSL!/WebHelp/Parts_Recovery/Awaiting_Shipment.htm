AWAITING SHIPMENT

The Awaiting Shipment folder contains details of parts that are marked to be shipped to the specified supplier location for recovery and reimbursement. The data in this folder allows you to efficiently manage your parts recovery.

 

To view the parts that need to be shipped to the supplier locations, click on the Parts Recovery tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the Awaiting Shipment folder.

 

The Awaiting Shipment table provides the following information:

  • Recovery Claim Number: The identification number of the claim to which the part to be returned belongs.

  • Product: The product name of the truck to which this part belongs.

  • Model: Model number of the truck to which this part belongs.

  • Status: The current status of the claim.

  • Part Number: Part number of the part that is to be returned.

  • Barcode: A bar code which uniquely identifies the part.

Double clicking on any claim will open a new page. The detail page is divided into various sections, Claim Details, Truck Details, Failure Information, Components Replaced, Part Return History, Recovery Cost Details, Supplier Recovery Credit Memo, Recovery History and Recoverable Parts details.
