pARTS CLAIM

You can submit a parts claim when the truck is out of warranty but the failed part is still covered by individual part warranty.

 

To create and submit a part claim:

When you click on the Create Claim option, a new page, New Claim is shown. You are now on Page 1 of 2 on this tab. You need to provide the following details on this page to submit a parts claim:

  • Dealer: This is a read-only field and displays the name of your dealership.

  • Use the Claim Type drop-down option to select the Parts option.

A parts claim can be serialized or non-serialized. If you have the serial number of the item, you can file a serialized claim. A non-serialized machine claim is filed when the serial number of the truck is not available on which you are submitting the claim or if the system fails to recognize the serial number.

  • Have part serial number:

  • If it is a serialized parts claim, select ‘Yes’ and choose any one of the following options.

Serialized_part_with_Part_installed_on_serialized_host_truck

Serialized_part_with_Part_installed_on_non-serialized_host_truck

Serialized_part_with_Part_installed_on_Competitor_Model

  • If it is a non-serialized parts claim, select ‘No’ and choose any one of the following options.

Non-serialized_part_with_Part_installed_on_serialized_host_truck

          Non-serialized_part_with_Part_installed_on_non-serialized_host_truck

Non-serialized_part_with_Part_installed_on_Competitor_Model         

Few of the fields in the New Claim page vary depending upon the selection of claim type that can be serialized or non-serialized part and the part installed selections. To view instructions for submitting the above types of part claim, click on the links above.
