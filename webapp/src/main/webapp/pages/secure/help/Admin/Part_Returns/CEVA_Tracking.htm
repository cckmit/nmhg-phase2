CEVA Tracking

When the dealer enters the shipment with load dimensions, then the claim is moved to CEVA Tracking inbox of the processor.

To view the CEVA Tracking inbox details, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the CEVA Tracking folder. On this page you can see the WPRA Number table table that lists all claims under this inbox.

Double clicking on a claim in this page opens it in a new page. The detail page is divided into two sections: Claim Details and Shipment Information for ease of navigation. The Claim Details section displays information about the claim to which the part belongs. The Shipment Details section will show information about the shipment.

 

You can see the following details in the Claim Details section, in the detail page:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Truck Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a field Product Improvement claim, this field will show the serial number of the machine on which you filed the Field Product Improvement claim.

  • Model Number: This field will show the model of the machine on which the claim was filed.

You can see the following details in the Shipment Information section:

  • WPRA Number:  The WPRA Number of the  part.

  • Tracking Number: The unique tracking number provided by the shipper for your shipment. This number allows you to track your shipment to its return location.

  • Shipment Date: Select the Date and time for shipping the consignment.

  • Carrier: Select the carrier that you used to ship the parts from the dropdown.

  • Load Type, Load Dimensions and Shipment Number: If you had entered the Load Information, then you can see the same in this field.

  • Comments: If you had added any additional comments about the shipment, parts being shipped, carrier used for the shipment, then you can see the same in this field.

Actions

You can perform the following actions on the shipment in this folder:

  • Submit: Clicking this button moves your claim to the Dealer's Part Shipped inbox. For details on this inbox, click here.

  • Cancel Return Process: If you want to cancel the Return Process, then click this button.
