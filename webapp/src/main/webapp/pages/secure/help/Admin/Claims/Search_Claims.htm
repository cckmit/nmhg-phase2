Search Claims

The Warranty Management application allows you to search for claims submitted by any service provider, using the Search Claims option.

 

To go to the search page,

  • Click on the Search Claims link under the Go To column on your Home page.

  • Click on the Claims tab on the left-hand pane. The tab will expand to show all folders under it. Click on the Define Search Query link.

Either of the above actions takes you to the Define Search Query page. You can search for claims based on various claim parameters on this page. You can also save a search for future viewing. To know more about creating or editing existing business conditions, click here.

 

Search Results

The results of your search will be shown in a tabular format. Click here to know more about the listing toolbar on the Search Results table. Click here to know more about navigating through the Search Results table.

 

Edit Search

You can edit a search phrase and modify any of the search parameters.

 

To edit a search phrase,

  • Click on the button on the search result page.

  • The Edit Search Query page is shown. You can see the search conditions created by you here.

  • You can edit any parameter in this field.

  • To reset the data in this page, click Reset. All details entered by you will be reset to the default selections.

  • To execute the modified search conditions, click Execute. The search results will be shown in a tabular format.

You can also search the claim using the PreDefined Search options listed under the Claims tab on the left-hand pane or click on Predefined Search (Claim) link under Common Actions of the Home page. To search for a claim, enter/select one or more parameters. Once you set the search parameter, perform one of the following actions:

  • If you wish to save this search, then check the Save Search check-box. The search will be saved in the Search Folders section under the search phrase specified by you.

  • If you want to clear the options selected by you, click on the Reset button.

  • If you want to search for a claim based on the condition you just created, click on the Search button. The page refreshes to show the search results in a tabular format.

If you know the claim number for which you want to view the details, click on Claim under Quick Search on the Home page. Just enter the claim number and click on View Claims. You can see all the details regarding the claim including the servicing location, truck details, failure and service information, payment and other details and the history linked to that claim.
