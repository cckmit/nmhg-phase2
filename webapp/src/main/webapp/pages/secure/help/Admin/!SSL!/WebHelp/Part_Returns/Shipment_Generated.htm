Shipment Generated

When you generate a shipment for a part or a group of parts, the details of the shipment are shown in the Shipment Generated folder. You can update the shipping details for each shipment here. You can also add/remove parts from a previously decided shipment, as per the dealer's request.

 

To view the shipment details, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the Shipment Generated folder. You can view this page in two views: Shipment View and WPRA View. You can toggle between these two views by clicking on the Shipment View and WPRA View buttons respectively.

WPRA View: This view displays the WPRA Number for the parts. Double click on an entry to display the Details page for that shipment.

Shipment View: This view displays the Shipment Number and Return Location of the parts. Double click on an entry to display the Details page for that shipment.

Note: You may need to collect relevant information from the dealer, about parts to be added/removed from a shipment, before generating new shipment tags. You will also need to collect the tracking number, carrier, freight amount, etc. details from the dealer before taking any actions on an entry in this folder. Since the dealer has no access to the Warranty Management system, the information exchange will have to be done off-line.

The detail page is divided into three sections, Claim Details, Part Return Details, and Shipment Information for ease of navigation. The Claim Details section displays information about the claim to which the part belongs. The Part Return Details will show all the parts that are to be returned to this location from the same claim. The Shipment Details section will show information about the shipment.

 

You can see the following details in the Claim Details section, in the detail page:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a field Product Improvement claim, this field will show the serial number of the machine on which you filed the Field Product Improvement claim.

  • Model: This field will show the model of the machine on which the claim was filed.

  • Dealer Job Number: The Dealer Job number assigned to the claim.

In the Part Return Details section, the following information is displayed in the detail page:

  • Select Checkbox: If you want to include this part in your shipment, then check this check-box.

  • Supplier Part Number: Supplier part number of the part that is to be returned.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • RGA Number: RGA Number of the part.

  • WPRA: WPRA Number of the part.

  • Return Directly to Supplier: Number of parts of the given specification that are to be returned directly to the supplier.

  • Pending Shipment: Number of parts of the given specification for which shipment is pending.

  • Shipment Generated: Number of parts of the given specification for which shipment is generated.

  • Cannot be Shipped: Number of parts of the given specification which cannot be shipped.

  • Shipped: Number of parts of the given specification that are to be shipped.

  • Total: The total number of parts.

  • Due Date: Date on which the part is due for return.

  • Due Days: Number of days between the current date and the due date for this part. If the part is not received within the specified days, then the claim will move under the Part Shipped Not Received inbox.

  • Overdue Days: Number days between the due date for this part and the current date. This data is shown only if you have not shipped the part within the suggested due date.

You can also add or remove a part to the list by clicking on Remove Parts from this Tag or Add New Part to this Tag.

Adding Parts to Shipment

You can add other parts which need to be returned to the same location, to this shipment and generate part tags for these parts.

  • To add new parts to the shipment,

  • Click on the Add Parts to Shipment button.

  • All parts that need to be returned to the same location as this shipment, but which are not in this shipment, will be shown. You can see the claim and part details for these parts.

  • Check the check-box against the part that you want to include in this shipment.

  • Click Add. All parts that you selected will be added to the shipment.

Removing Parts from Shipment

If you want to remove any parts from this shipment, clear the check-boxes against those parts and click Remove Parts Added to the Shipment button. The selected parts will be removed from the shipment.

 

You will need to provide the following details in the Shipment Information section:

To enter the Load Type, click on the option.

  • Load Type: Select the desired load type from the drop-down.

  • Load Dimensions: Enter the load dimensions.

  • Add to shipment: Select the desired shipment from the drop-down

You can delete a load type by clicking the option corresponding that row.

  • Tracking Number: Enter the unique tracking number provided by the shipper for your shipment. This number allows you to track your shipment to its return location.

  • Shipment Date: Use the option in this field to enter the date of shipment. You can also manually enter the date in this field.

  • Carrier: Use the drop-down box to select a shipper through which you are shipping the parts.

  • Comments: If you have any additional comments about the shipment, parts being shipped, carrier used for the shipment, then you can enter the same in this field.

Actions

You can perform the following actions on the shipment in this folder:

  • Print Shipment Tag: If you have added parts to your shipment or removed parts from it, then you can re-generate the shipment tag and the part tags for new parts by clicking this button. Click here to know more about the shipment tag.

  • Submit: If you want to submit this shipment, then click this button. This indicates that you have shipped the parts included in this shipment to the specified location. A success message will be shown confirming your action.
