SEARCH Parts

You can also search the part using the PreDefined Search options listed under the Part Recovery tab on the left-hand pane or click on the Predefined Search (Recovery Part Return) link under the Common Actions on the Home page. To search for a part, enter/select one or more parameters.

  • Supplier Name: The name of the supplier of the part.

  • Supplier Number: The identification number of the supplier.

  • Part Number: Part number of the part that is to be returned.

  • Claim Number: The identification number of the claim to which this part belongs.

  • Tracking Number: Tracking number of the part returned.

  • Status: Select the status of the part from the drop down list.

Once you set the search parameter, perform one of the following actions:

  • If you wish to save this search, then check the Save Search check-box. The search will be saved in the Search Folders section under the search phrase specified by you.

  • If you want to clear the options selected by you, click on the Reset button.

  • If you want to search for a part based on the condition you just created, click on the Search button. The page refreshes to show the search results in a tabular format.
