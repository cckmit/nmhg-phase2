MANAGE business rule GROUPS

If the claim is satisfying more than one business condition, then the system will check for the priority and route it to the processor. Manage Business Rule Groups helps in managing the rules by changing the priority of the business conditions.

 

You can create rules for,

  • Claims Processing

  • Claim Processor Routing

  • DSM Routing

  • Advisor Routing

Click on Manage Business Rule Groups under the Warranty Admin tab on the left-hand pane. Click on an individual link to view and create a business rule group. The right-hand pane lists all the rule groups created by you. For information on navigating through this table, see Lists.

 

You can add new rule group or edit existing rule groups as per your need.

 

To create a rule group,

  • Click on Create Rule Group button in the listing toolbar.

  • Enter the name of the rule group you wish to create.

  • Select the checkbox for 'Stop Rule Execution If At Least One Rule Fires' or 'Stop Rule Execution If And When A Rule Fires' as per your requirement. You can select either one or both of the check boxes.

  • Click Create.

A new rule group is created.

 

To edit a rule group,

You can edit a rule group by opening it in the detailed page. You can adjust the priority of rules by clicking on a specific rule and using drag and drop to move the rule to the desired location. Click Update to save the changes.

 

Re-prioritize rule groups

You can also re-prioritize rule groups as per your requirement.

To re-prioritize rule groups, click on Re-prioritize Rule Groups button in the listing toolbar. Use drag and drop to adjust the priority of the rule groups. Click Update to save the changes.
