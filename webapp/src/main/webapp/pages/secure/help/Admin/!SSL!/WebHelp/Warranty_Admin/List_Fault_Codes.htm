List Fault LOCATIONS

The List Fault Locations page lists all fault codes created by you. The Warranty Management application allows you to associate labels to groups of fault codes.

 

To view the list of fault codes, click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Failure Structure folder. All links under this folder are shown. Click on the List Fault Locations link. On the right-hand pane, you can see the Fault Codes table that lists all fault codes created by you. See Lists for more information on navigating through this table.

 

The Fault Codes table provides the following information about each fault code:

  • System: Items in this column form the first level parent node in a failure hierarchy tree. These are the parts/systems that constitute the equipments. For example: Engine, Cylinder, etc.

  • Sub-system: This is the second level in the failure hierarchy tree. Items in this column form the child nodes of the part/system. These are the sub-parts constituting the corresponding part/system. For example: If the parent node is Engine, the sub-system node can be Injector Circuit, Position Sensor, etc.

  • Component: This is the third level in the failure hierarchy tree. Items in this column are the child nodes of the corresponding sub-parts. These are the components constituting the corresponding sub-parts. For example: If the parent node is Engine-Position Sensor, the component can be Control Valve.

  • Sub-component :This is the fourth and last level in the failure hierarchy tree. Items in this column are the leaf nodes. These are the sub-components constituting the corresponding components. For example: If the parent node is Engine-Position Sensor-Control Valve, then the sub-component can be <>.

  • Labels: This column displays the icon when no labels are applied on the fault code. When a label is applied on the fault code, the icon changes to . If you place your mouse cursor on this icon, the names of labels applied on the fault code will be displayed.

To associate a label with a group of fault codes,

  • Click on a fault code to select it. You can select multiple fault codes by pressing the Ctrl key and simultaneously clicking the fault codes.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to apply. This field supports auto-suggest for all previously created labels. You can also create a new label by entering its name in this field.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To apply the selected labels, click Apply Label(s). The selected labels will be applied to the selected fault codes.

To remove a label from a group of fault codes,

  • Click on a fault code to select it. You can select multiple fault codes by pressing the Ctrl key and simultaneously clicking the fault codes.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to remove. This field supports auto-suggest for all previously created labels.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To remove the selected labels, click Remove Label(s). The selected labels will be removed from the selected fault codes.
