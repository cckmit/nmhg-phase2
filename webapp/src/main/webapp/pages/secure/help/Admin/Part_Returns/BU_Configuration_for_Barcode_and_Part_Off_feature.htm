BU CONFIGURATION FOR BARCODE AND PART OFF FEATURE

BU Configuration for Barcode

The barcode field appears for the dealer only when the Administrator selects Yes for Enable Barcode Feature in the Manage Business Configurations page. Thus it enables the dealers to enter barcodes while returning parts.

 

Perform the following steps to configure the barcode feature for the dealer:

  • Click on Manage Business Configurations tab under the Warranty Admin tab on the left-hand pane.

  • Click on Return Part Management under the Claims section on the right hand pane.

  • Select the radio button Yes for Enable Barcode feature.

  • Click Update.

  • This message is displayed: BUSINESS UNIT PARAMETERS HAVE BEEN UPDATED SUCCESSFULLY

BU Configuration for Part off feature

Sometimes the receiver wants to edit the part number entered by the dealer. If the Part off feature is enabled by you, the Part Receiver can correct the part number and part serial number entered by the dealer.

 

Perform the following steps to configure the Part off feature for the dealer:

  • Click on Manage Business Configurations tab under the Warranty Admin tab on the left-hand pane.

  • Click on Return Part Management under the Claims section on the right hand pane.

  • Select the radio button Yes for Enable Part-off feature.

  • Click Update.

  • This message is displayed: BUSINESS UNIT PARAMETERS HAVE BEEN UPDATED SUCCESSFULLY.

Part off Enhancements

Sometimes the receiver wants to edit the part number entered by the dealer. If the Part off feature is enabled by you, the Part Receiver can edit the part number and part serial number entered by the dealer.

To configure the Part off feature for the dealer, see BU Configuration for Part off feature .

Thus the modifications done by the receiver are reflected in the processor review inbox of the processor. This is known as part off enhancement. To configure the part return list visibility to dealers, click here.
