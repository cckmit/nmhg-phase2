MACHINE CLAIMS

You can submit a machine claim when the truck is covered by warranty policies.

 

To create and submit a machine claim:

When you click on the Create Claim option under Go To on the Home Page, a new page, New Claim is shown. You are now on Page 1 of 2 on this tab. You need to provide the following details on this page to submit a machine claim:

  • Business Unit: This is a read-only field and displays the name of your Business Unit.

  • Dealer: This is a read-only field and displays the name of your dealership.

  • Claim Type: In this drop-down field, select the Machine Claim option.

  • Truck Serial Number: If you have the serial number of the truck on which you are submitting the claim, enter the serial number. This number will uniquely identify the truck on which the claim is being filed and also determine the warranty coverages associated with the truck.

  • Dealer Job Number: Enter your internal tracking number for this claim.

  • Hours On Truck: Enter the hours on truck when the part was originally installed.

  • Date of Failure: Click on the option to specify the date on which the truck failed. You can also type the date manually in this field.

  • Repair Start Date: Click on the option to specify the date on which the truck  repair started. You can also type the date manually in this field.

  • Repair End Date: Click on the option to specify the date on which the truck  repair ended. You can also type the date manually in this field. Note that if you enter Repair End Date before the Repair Start Date and click Continue you will get the following error:

Please enter a repair start date on or before repair end date.

  • Contact Management Ticket Number: Enter the Contact Management Ticket Number on the claim here. This ticket number is linked to the ticket detail screen in Contact Management system.

  • NCR: Machine shipment and machine arrival condition issues are handled through NCR (Non-Conformance Report) claims. The Processors have an option to file NCR claims. Select this checkbox if you want to file a NCR claim. These claims can be filed only for Stock inventories.

  • NCR With 30 Days: The Processors and some of the dealers have an option to file NCR with 30 days claims. Select this checkbox if you want to file a NCR claim with 30 days. These claims can be filed only for Retailed inventories.

Note: If the Claim Date is less then the shipment date or if the Claim Date is greater than Shipment Date by 6 months, then the Processor won’t be allowed to file the NCR claim.

  • Request SMR: If you want to submit the claim for a review by the District Service Manager (DSM), then select this checkbox. This is also called pre-authorization. The claim, on submitting, will be sent to the district service manager associated with the dealership to which the dealer belongs.

  • SMR Reason: This field is shown only when you select the Request SMR check-box. Use this drop-down field to specify the reason why you are submitting the claim for a review by the DSM.

After you have entered all relevant details in the above fields, click Cancel if you do not wish to proceed with the claim submission. This will close the New Claim page without saving any details entered by you.

To move to the next step of claim submission, click on Continue >>. This will also save your claim in the Draft folder.

You are now on Page 2 of 2 of the New Claim page. This page is divided into various sections for ease of navigation. The fields under some sections on this page are hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses it hiding all fields under it.

 

Claim Details

This is a read-only section shown on top of Page 2 of 2. This section provides you information about the claim entered by you on page 1.

This section also displays three fields as links. These are explained below.

Contact Management Ticket Number: The User Management ticket Number which you have entered in claim Page 1 appears here as a hyperlink. Clicking on this link redirects you to the Contact Management Detail screen.

Dealer: Click on this link to display the Dealer Information window. It displays the complete dealer details: his name, address, city, state, country and postal zip code.

Owner Name: Click on this link to display the Owner Information window. It displays the complete Owner details: his name, address, city, state, country, postal zip code and type.

If you have not entered the Owner information previously, the link Enter Owner Information is displayed instead of the link Owner Name. Click on this link to enter the Owner Information.

 

Dealer/Servicing Location

Select the information regarding the servicing location of the customer from this dropdown.

 

Truck Details

This is also a read-only section and will provide you with the following details about the truck on which you are filing a claim:

  • Truck Serial Number: Serial number of the truck. This is a hyperlink to the Truck Details page of the truck. For further details on the truck details page, click here.

  • Item Number: Item number of the truck.

  • Item Description: A short description of the truck. If the description is truncated, mouse over on the text to view the entire description.

  • Product: Product name of the truck.

  • Model: Model of the truck.

  • Hours On Truck: This field shows the same number of hours which the dealer has entered  on claim page 1 for the field Hours On Truck.

  • Warranty Start Date: Date when the warranty coverages commenced on this truck. This is the earliest start date among all policies associated with this truck. This field will appear only in the case of serialized machine claim.

  • Warranty End Date: Date when this truck will be out of warranty. This is the latest end date among all policies associated with this truck. This field will appear only in the case of serialized machine claim.

  • Additional Attribute: Any additional information associated to the truck will auto-populate.

Failure Information

This section captures relevant information regarding the failure in the truck. You need to provide the following details here:

  • Causal Part Number: Specify the part number of the part which caused the failure. This field supports auto-complete. As you enter the causal part number in this field, the application prompts the possible values in a drop-down list. You can select the appropriate part number by clicking on it.

  • Causal Part Description: It auto-populates.

  • Click on the Fault Location link. You can see a tree like hierarchy of fault location on this pop-up. Select the fault location and click on Set Fault Location.

  • Fault Found: The fault found captures the nature of the failure that happened. Use this drop-down field to select the appropriate fault found in the truck.

  • Caused By: Use this drop-down field to select the appropriate cause of fault found in the truck.

  • Failure Detail: Use this drop-down field to select the failure detail in the truck. Click on the link View Unit Diagram to view the specific assembly BOM/diagram for this truck.

Alarm Code

Alarm Codes are used to give additional information about the product while filing a claim.

  • Alarm Code: Select the name of the alarm code from the drop-down.

  • Description: A short description of the alarm code will auto-populate.

Service Information

In this section you can specify all the services that were performed for the claim.

 

Service Procedure

A service procedure is identified using the job-code associated with it. Just like a fault code, the job code allows easy identification of the labor performed on the truck.

 

To specify a labor detail,

  • Click on the option.

  • The Select Service Procedures pop-up is shown. You can see a tree like structure of service procedures here.

  • Click on the icon in front of a service procedure to view the entire structure.

  • Check the check-box against the service procedure you want to select.

  • You can select multiple service procedures on this window.

  • Now click on the Add Service Procedure button. All service procedures selected by you will be shown as different rows under the Service Information section.

You can view/enter the following details for each service procedure:

  • Job Code: The service procedure selected by you. A mouse over on the icon alongside the job code will show a short description of the service procedure.

  • Standard Labor Hour(s): Time, in hours and minutes, required to perform the labor. This is the standard time required for the labor.

  • Additional Labor Hour(s): In addition to standard labor time, you have the option to enter additional labor hours, if any, for the labor. This field will appear for only those dealers who have been added to the eligibility list by the admin.

  • Reason for Additional Labor Hour(s): Specify the reason for consuming additional time for the labor. This field is shown only when you enter additional labor hours.

  • Additional Attribute: This is an optional field. If an attribute is associated to the truck, you will see Enter Additional Attribute link where you can add any additional information here which may be text, date or number depending upon the data type selected while creating the attribute.

You can delete a service procedure by clicking the option corresponding that row.

 

Labor Split

Sometimes the repair/service may require labor to work beyond the normal working hours and hence eligible for overtime pay.

To enter the labor split information, click on the option.

  • Over Time Labor Split: Select the defined labor split from the drop-down.

  • Labor Hours(s): Enter the number of hours the labor was performed beyond the normal working hours.

  • Reason for Over Time Labor Hour(s): Enter the reason for performing the overtime labor hours.

You can delete a service procedure by clicking the option corresponding that row.

 

Components Replaced

The Components Replaced section has three sub-sections: OEM Parts Replaced Installed, Miscellaneous Expense & Outside Services and Miscellaneous Parts.

 

OEM Parts Replaced Installed

In this sub-section you can specify details on all components that were installed/removed for fixing the failure.

Note: It is mandatory that the causal part be a part of the Parts Replaced sub-section for a claim.

To enter details for a Installed or Replaced part, click on the option. A new row is added to this sub-section. Here, you need to mention the following information for each part:

  • Component Serial Number: If you have removed a serialized part, then you would have replaced it with another serialized part of the same kind. Enter the serial number of the newly installed part in this field.

  • Part Number: Enter the part number of the installed/removed part.

  • Quantity: Enter the quantity of parts of the specified kind that needed replacement.

  • Description: This is a read-only field and will show a short description of the part specified by you.

  • Failure Report: This field shows a short description of the part specified by you.

You can enter multiple parts at a time in this sub-section by clicking the option. To delete a row, click on the option corresponding to that row.

 

Miscellaneous Expense & Outside Services

In this sub-section, you can claim expenses related to any non OEM services or parts that the dealer used for fixing the failure. To enter details for a part/service, click on the option. A new row is added to this sub-section. In a row you need to mention the following information for each part/service:

  • Description: Provide a detailed description of the non OEM service/part that you are claiming.

Note: If you enter any OEM Part number in Miscellaneous Expense & Outside Services Part Description, then the system displays this error message "You have entered OEM Part number in NON OEM Part Description. Please enter OEM Part number under OEM Parts section."

  • Quantity: Enter the quantity for the service/part.

  • Unit Price: Enter the unit price for the service/part.

  • Invoice: For any outside part/service used, it is mandatory to provide an invoice as proof of expense. To attach an invoice,

  • Click on the Attach Invoice link. A pop-up window opens.

  • Use the Browse button to browse through your system for the soft-copy of the invoice.

  • After providing the proper link to the invoice file, click Attach. The selected invoice file will be attached to your claim.

To delete a row, click on the option.

 

Travel Details

In this section, you can enter the travel information of the dealer:

  • Travel Distance: This field is autopopulated based on Google maps calculation.

  • Travel Location: Click on the Google maps link. This pre populates From address as Dealer location or remote technician location and To address as customer address. Edit the To Address,  if desired and click Submit.

  • Additional Travel Distance: Enter the additional travel distance, here.

  • Reason for Additional Travel Distance: Enter your reason for the additional travel distance.

  • Travel Trips: Enter the number of travel trips to the specified location.

  • Travel Hours: This field is autopopulated based on Google maps calculation.

  • Additional Travel Hours: Enter the additional travel hours, here.

  • Technician: Use the drop-down to specify the technician who performed this labor. This field is optional.

Miscellaneous

Here, the section Incidentals allows you to enter the other related expenses such as Item Freight and Duty, Parking and Toll expense, Meals, etc.

 

Claim Description

This section allows you to provide a detailed description of the fault found in the truck, cause for the failure of the truck and the repair works performed to correct the failure.

  • Condition Found: Use this text field to provide a detailed description of the failure found in the truck.

  • Work Performed: Use this text box to provide a detailed description of the repair work performed to correct the failure.

  • Claim Notes: This is an optional field. You can provide any additional comments like additional labor, travel involved in this text box.

Manage Documents

In this section you can attach binary documents, provided by the dealer, to support any of expenses and claims or the nature of failure in the truck. To attach a document, click on the button. A pop-up window is shown. Click on the Browse button to browse through your system for the document. You can attach up to 20 documents for a claim. After you have mentioned the exact location of the documents, click Attach. All selected documents will be attached to the claim. To delete an entry, click on the option.

 

Actions

After providing necessary details in all mandatory fields, you can take one of the following actions on the claim:

  • Cancel: To close the New Claim page without saving any details entered on Page 2 of 2 after the last save.

  • Go to Page 1: To  go to Page 1 of 2. You can edit the data entered on page 1.

  • Delete: To delete this claim from the Draft folder.

  • Save: To save all details entered by you in Page 2 of 2. The claim will be saved in the Draft Claim folder.

  • Validate: To validate the claim and submit it. For further information about claim validation and submission, click here.

Depending on the action taken by you, a success message will be shown confirming your action. The message will hide itself and the current page will be closed.
