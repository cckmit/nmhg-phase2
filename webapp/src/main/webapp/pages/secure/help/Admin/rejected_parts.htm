Rejected PARTS

Dealers can claim the shipped parts back from NMHG lab, if the claim has been denied or the parts have been rejected by the inspector or supplier (if parts have been shipped to supplier) and warranty claim is also denied.

If the claim is denied, then the Warranty processor may flag the parts to be returned to dealer and dealer is notified for the same in his Rejected Parts inbox.

To view the entire list of claims which have been rejected or if parts have been rejected for a claim,

  • Click on the Rejected Parts link under the Action Folders column on your Home page.

  • Click on the Part Returns tab on the left-hand pane. The tab opens to display the folders under it. Click on the Rejected Parts folder.

Either of the above actions takes you to the Rejected Parts page. On this page you can see all the rejected claims which have some returnable parts. Double click on a claim to display the details of the rejected parts.

Select/Unselect All Parts

Select this checkbox if you want to select all the parts together which appear in the Rejected Parts Details section.

This page contains the following sections:

Claim Details

You can see the following details in the Claim Details section, in the detail page:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a Field Product Improvement claim, this field will show the serial number of the machine on which you filed the Field Product Improvement claim. The serial number is also a hyperlink to the truck details page. Click here for more information on the truck detail page.

  • Model Number: This field will show the model number of the machine on which the claim was filed.

Rejected Parts Details

You can see the following details in the Rejected Parts Details section, in the detail page:

  • Part Number: Part Number of the part that is rejected.

  • Description: A short description of the part that is rejected.

  • Shipped: The quantity of the parts shipped.

  • Request for Return: The quantity of the parts requested for return.

  • Return Not Required: The quantity of the parts whose return is not required.

  • Return Part Status: The return part status of the returned parts.

Comments

If you have any additional comments about the rejected parts, then you can enter the same in this field.

 

Request For Parts Back

If the part return process has not been initiated, then you can raise the Part Return Request by clicking on the Request for Parts Back button. After you have submitted the claim and requested for the parts, it moves to the part shipper’s “Parts Return to Dealer -> Dealer’s Requested Part” inbox.

 

Note: The dealer must request for the parts within 30 days. This button is disabled after 30 days.
