Shipment TAG

A shipment tag is generated when you click on the Generate Shipment button in the Due Parts and the Overdue Parts folders. The number of shipment tags generated is equal to the number of unique return locations. The shipment tag also includes separate tags for each part.

 

Warranty Claim Summary

The Warranty Claim Summary is displayed in the first page. It includes the following details:

  • Admin Configured Message: The message configured by the Admin for shipping the requested part.

  • Part Number: Part Number of the Part being returned.

  • Description: Description of the Part being returned.

  • Return Due Date: Due date for the Part to reach the return warehouse location.

  • Failure Date: Failure Date of the part.

  • Claim Number: The claim number of the claim filed on the part being returned.

  • RMA/RGA Number: The RMA/RGA number that is set up by the Warranty Admin/Part Return Admin during the Part return configuration or field product improvement campaign setup.

  • M/C Serial Number: The machine serial number for the part as entered during the claim creation.

  • Model: The product model number of the part being returned.

  • Dealer Job Number: Dealer Job number of the claim filed on the part being returned.

  • Receiver Instructions: The receiver instructions provided by the Admin while setting up the Part Return configuration.

  • Claim Comments: The claim comments describe the damage and repair for the Part being returned as below.

  • Fault: The condition found comments entered on the claim.

  • Cause: The cause comments entered on the claim.

  • Work Performed: The work performed describes the necessary action taken by the dealer to correct the machine caused by damaged part as entered during claim filing.

  • Additional Notes:  The Claim Notes entered during claim filing.

  • Date Part Return Was Requested By Supplier: The date by which Part Return is requested by the supplier.

  • Dealer Info: The complete information about the dealer.

  • Problem Description: A short description about the problem in the part.

  • Customer Name: The name of the customer.

  • Problem Part Number: The part number of the part causing the problem.

  • Vendor Part Number: The part number of the vendor.

  • Hour Meter: Value in the Hour Meter.

  • Installation Date: The installation date of the part.

Shipment Tag

  • From: This section shows the Dealer's Name, complete postal address and Dealer Number.

  • Return to Address: The address to which the shipment is being done. This is the complete postal address of the location where you have been asked to return the part(s).

  • Sender Comments: The comments of the sender, if any.

  • Print Date: The print date and time of the shipment tag.

  • Type of Load: This field displays the shipment load type.

  • Load Dimension: This field displays the shipment load dimension data.

  • Shipment Number: The shipment number of this shipment.

  • WPRA Number: The WPRA number for this shipment.

  • Shipment Date: The date and time of the shipment.

  • Tracking Number: The tracking number of the shipment.

List of Parts to be Returned

You can see this list as a part of the shipment tag.  You can see the following details in this list:

  • Claim Number: The number of the claim to which this part belongs.

  • Model Number: The product model number of the part being returned.

  • Serial Number: Serial number of the machine or the part, depending on the claim type, on which you filed the claim.

  • Part Number: Part number of the part that is being returned.

  • Description: Description of the Part being returned.

  • Quantity: Quantity of the Part being returned.

  • RMA/RGA Number: RMA/RGA Number of the Part being returned.

  • Country of Origin: The country of origin of the part.

You need to cut each part tag and paste it on the package of that part before placing it in the shipment.
