Confirm Dealer Part Returns

Instead of part being shipped to NMHG warehouse, processors can change the part return location (not to the warehouse set on PRC or other warehouses) on claims to send part(s) directly to supplier (Supplier location). Thus, when the processor asks the dealer to ship the parts directly to the vendor/supplier, this part comes to the Confirm Dealer Part Returns inbox of the supplier. This is only for parts with no PRC configured.

The claim status remains “on hold” till the supplier receives the part.

You can see the following details for each part in the Confirm Dealer Part Returns list page:

  • Shipment Number: Identification number for the consignment.

  • Dealer: This column specifies the name of the dealer for this part.

  • Carrier: Name of the shipper you used for shipping the parts to the return location.

  • Tracking Number: Tracking number for your shipment provided by the shipper.

  • Shipment Date: Date and time when the parts were shipped.

Double click on a part to view the following Part Details:

  • OEM Part Number: The OEM Part Number of the part.

  • Supplier Part Number: The Supplier Part Number of the part.

  • Description: The description of the part.

  • Quantity: Quantity of the part to be returned.

  • RGA Number: RGA number of the part to be returned.

  • Action: Select the desired option from the two options available: Received and Not Received. Click Submit to submit the action taken by you on this part. A success message will be shown confirming your action.

If you have selected the option Received and clicked Submit, it means the part has been received by the supplier.  Thus, its status changes to Part Received by supplier. You can see this part status in the Part Return History of the part. For details on the Part Return History, click here.
