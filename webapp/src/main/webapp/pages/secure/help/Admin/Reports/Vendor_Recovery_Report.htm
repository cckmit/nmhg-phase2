VENDOR RECOVERY REPORT

The Vendor Recovery Report feature allows you to generate a report for a particular recovery claim status. The report lists all the claims with a selected status.

 

Perform the following steps to generate a vendor recovery report:

  • Click on Vendor Recovery Report under the Reports tab on the left pane.

  • Enter the date range for which you want to extract the created recovery claims.

  • Select the Status of the claim for which you want the list.

  • Click Submit.

The data for the selected period and status will be listed in a xls file which you can save on your desktop for future reference.
