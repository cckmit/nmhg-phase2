List Service Procedures

A service procedure is an extension to the fault location except that it indicates the action that is performed as part of the repair. It is also referred to as the job code. The List Service Procedures page lists all service procedures created by you. The Warranty Management application allows you to associate labels to groups of service procedures.

 

To view the list of service procedures, click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Failure Structure folder. All links under this folder are shown. Click on the List Service Procedures link. On the right-hand pane, you can see the Service Procedures table that lists all job codes created by you. See Lists for more information on navigating through this table.

 

The Service Procedures table provides the following information about each job code:

  • System: Parts/systems that constitute the trucks.

  • Sub-system: Sub-parts/sub-systems constituting the corresponding part/system.

  • Component: Components constituting the corresponding sub-parts/sub-systems.

  • Sub-component: Sub-components constituting the corresponding components.

  • Action: This column shows the service procedure defined for each fault code.

  • Labels: This column displays the icon when no labels are applied on the job code. When a label is applied on the job code, the icon changes to . If you place your mouse cursor on this icon, the names of labels applied on the job code will be displayed.

To associate a label with a group of job codes,

  • Click on a job code to select it. You can select multiple fault codes by pressing the Ctrl key and simultaneously clicking the job codes.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to apply. This field supports auto-suggest for all previously created labels. You can also create a new label by entering its name in this field.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To apply the selected labels, click Apply Label(s). The selected labels will be applied to the selected job codes.

To remove a label from a group of job codes,

  • Click on a job code to select it. You can select multiple fault codes by pressing the Ctrl key and simultaneously clicking the job codes.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to remove. This field supports auto-suggest for all previously created labels.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To remove the selected labels, click Remove Label(s). The selected labels will be applied to the selected job codes.
