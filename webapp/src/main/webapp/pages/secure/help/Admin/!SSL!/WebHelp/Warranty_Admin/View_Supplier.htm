VIEW SUPPLIER

A supplier is a person representing the manufacturer who supplies parts to the OEM.

 

You can view a list of suppliers by clicking on View Supplier tab under the Warranty Admin tab on the left-hand pane. It is also present under the Contract Admin tab on the left-hand pane. On the right-hand pane, you can see the Supplier table. See Lists for more information on navigating through this table.

 

The Supplier table provides the following information about each supplier:

  • Name: The name of the supplier.

  • Supplier Number: It is the unique number associated with the supplier.

  • Address, City, State, Country, Postal code: These details are specific to a supplier.

  • Labels: This column displays the icon when no labels are applied on the supplier. When a label is applied on the supplier, the icon changes to . If you place your mouse cursor on this icon, the names of labels applied on the supplier will be displayed.

To associate a label with a group of suppliers,

  • Click on a supplier name to select it. You can select multiple suppliers by pressing the Ctrl key and simultaneously clicking the suppliers.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to apply. This field supports auto-suggest for all previously created labels. You can also create a new label by entering its name in this field.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To apply the selected labels, click Apply Label(s). The selected labels will be applied to the selected suppliers.

To remove a label from a group of suppliers,

  • Click on a supplier to select it. You can select multiple suppliers by pressing the Ctrl key and simultaneously clicking the suppliers.

  • Click on icon. A pop-up window is shown.

  • In the Label field, enter the name of the label that you wish to remove. This field supports auto-suggest for all previously created labels.

  • Click Select to add the label to the Selected Labels list.

  • You can add multiple label names in the same manner.

  • To remove a label from this list, enter the name of that label in the Label field and click Clear.

  • To remove the selected labels, click Remove Label(s). The selected labels will be removed from the selected suppliers.
