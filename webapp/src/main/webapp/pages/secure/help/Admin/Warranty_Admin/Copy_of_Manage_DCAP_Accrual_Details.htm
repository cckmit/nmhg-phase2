MANAGE DCAP ACCRUAL DETAILS

A dealer can file a DCAP claim only on those inventories that are listed on Manage DCAP Accrual Details page.

 

To view the list of inventories, click on Manage DCAP Accrual Details tab under the Warranty Admin tab on the left-hand pane. This page lists the serial number of the inventories on which you can file a DCAP claim. It also displays the DCAP accrual amount set under SCA and SEA. This accrual amount is selected from the price matrix based on Dealer Category, DCAP Model code, currency and the shipment date. If there is no price setup, the accrual amount is set to zero. However, Admin can change this amount. To know more about price matrix, click here.

 

To view and update the accrual details, double click on a particular serial number. The Manage DCAP Accrual Details page displays the accrual details and DCAP claim details for the serial number. You can also update the accrual details by updating the editable fields. Once you have updated, click on Update to save the details.
