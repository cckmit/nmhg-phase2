MANAGE MINIMUM LABOUR ROUND UP

Labour round up is applicable to the jobs codes that have labour hours associated to them.

 

To configure minimum labour round up utility,

  • Click on the Manage Minimum Labour Round Up tab under the Warranty Admin tab on the left-hand pane or click on the Manage Minimum Labour Round Up link under Common Actions on the Home page. Enter the details on the Configure Minimum Labour Round Up page.

  • Days between repair: Enter the number of days for repair. This field calculates the window period within which the repair date should fall.

  • Round up hours: Enter the maximum number of hours to which labour hours can be rounded up.

  • Applicable to commercial policy: Select ‘Yes’, if you want the labour round up utility set for commercial policy, else, select ‘No’.

  • Applicable to claim types: Select the claim type (Machine/ Parts/ Field Modification) for which you want to configure labour round up.

  • Click on the  button against Specify Products to add the products to which you want to associate the additional labour round up utility. You can add multiple products by clicking on the button.

  • Click Save.

Once the claim is approved, and the total labor hours including additional labor hours is less than the limit set up by admin, the system rounds up the labor to the minimum set up if it finds that no other claim has been made on the unit by the same dealer. When another claim is approved, the system checks to see if the claim falls within the repair window period of the previous claim. If it does, the system adjusts the labor hours on the new claim making sure that the collective labor hours is as set up by the admin.
