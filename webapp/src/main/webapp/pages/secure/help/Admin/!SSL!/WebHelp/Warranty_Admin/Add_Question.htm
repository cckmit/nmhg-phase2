ADD QUESTION

The Add Question link appears when the report is saved. This section allows to add questions regarding the product or information on the product.

 

To Add Question, perform the following -

  1. Enter the question in the Question text box. This has limit of 4000 characters.

  2. Enter the pre instructions in the Pre-Instruction text box. This has limit of 4000 characters. You can also upload an image in Pre-Instructions section. Click Browse and choose the image to upload.

  3. Select Answer Type from the following options -

  • Small Text

  • Large Text

  • Number - Allows number in the text field.

  • Single-select options (radio)

  • Multi-select options (checkbox)

  • Single-select list

  • Multi-select list

  1. Include Other as an Option - Select this option to include the “Other” option. If the administrator includes this option and the claimant selects it, a text area is displayed to enter any additional details.

  2. Enter the post instruction details in the Post-Instruction text box. This has limit of 4000 characters. You can also upload an image in Pre-Instructions section. Click Browse and choose the image to upload.

  3. After entering all the details click Save or Add Questions.
