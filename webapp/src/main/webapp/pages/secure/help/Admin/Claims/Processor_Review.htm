NEW

When a claim is first assigned to you for manual adjudication, it appears in your New inbox. You can make any changes to the claim details submitted by the service provider, except the claim type, the truck details and SMR Request/Reason data entered by the service provider.

 

To view all claims assigned for processor review,

  • Click on the link New under Action Folders column on your Home page.

  • Click on the Claims tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the New folder.

Either of the above actions takes you to the New inbox page. This page also displays two dropdowns for creation of custom views:

Actions: This dropdown enables two actions- Creation of a new view and edit selected view.

To create a new view, click Create View from the Actions dropdown. The Create Inbox View page is displayed. It displays the following fields:

  • View Name: Enter the name of the view.

  • Available Fields: All the fields available or creating a view are displayed in this section such as Acceptance Reason, Amount Asked, Amount Credited, Assign To etc. You can select a maximum of 12 fields together for creating a view.

  • Selected Fields: All the fields selected by you are displayed in this section.

  • Default Sorting Criteria: Select the default sorting criteria from this dropdown.

  • Sorting Order: Select the sorting order as either ascending or descending.

After entering all the details, click Submit to create this view.

 

View: This dropdown displays all the views available. Select the desired view from the dropdown. This can be either the already existing view or the view which you have created. The table displayed in this page displays the fields present in the selected view. By default, the default view is displayed.

 

You can see the following details for each claim in this table in the default view:

  • Claim Number: Reference number of the claim.

  • Dealer Name: The name of the Dealer for this claim.

  • Claim Type: The type of claim.

  • Causal Part: The Causal Part name for this claim.

  • Failure Date: The failure date of the part.

  • Repair Date: The repair date of the part.

You can view the details of any claim by double clicking on it. This opens it in a new page, <claim number>. The claim detail page for a claim in this folder, is divided into various sections for ease of navigation: Errors, History, Claim Details, Dealer/Servicing Location, Failure Information,Labour Information, Components Replaced, Travel Details, Miscellaneous, Comments, Part Return History, Manage Documents, Payment Details, Credit Memo History, Supplier Information and Actions. The fields under some sections on this page may be hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses it, hiding all fields under it.

 

For further information on Claim History, Claim Header, Truck Details, Failure Information, Service Information, Components Replaced, Miscellaneous, and Manage Documents click here.

 

Errors

This section displays all processing rule failures that occurred on this claim. These are the rules whose failure led to the claim being assigned to you for manual processing. You can see the description of each failed rule here.

 

Payment Details

In this section you can see complete details of the claim expenses broken down by different cost categories. The following details are shown as part of this section:

  • Instructions: The following instruction is displayed here. "Please note that Percentage Accepted takes effect only if you decide to accept this claim (in the Decision section below)."

  • Payment Details: The following columns are displayed here:

  • Category: Name of the cost category.

  • Amount Asked: Amount asked by the service provider for this cost category.

  • Current Amount: Current claim amount for this cost category. This amount may or may not be equal to that claimed by the dealer.

For example, if you changed the labor hours for this claim, then the amount shown in the Current Amount field will be different than the amount shown in the Amount Asked field for Labor Hours cost category.

  • Flat Amount/Percentage Accepted: This is an editable field and you can enter the amount in two ways: Either percentage of the total amount for this cost category or the flat amount in dollars that will be paid to the dealer for this claim. You can toggle between Flat Amount and Percentage Accepted by clicking on that option and then clicking on Click here to refresh the payment section. You can apply the percentages either at the totals of each individual cost category or at the total claim amount level and not at both levels.

  • Accepted Amount: The final amount that is calculated after applying the accepted percentage entered by you.

Comments

You need to enter comments in the following fields for a claim in the New folder:

  • Internal Comments: The comments that you enter in this field are visible only to other OEM users.

  • Comments to Dealer: The comments that you enter in this field are visible to both the OEM users as well as to the dealer who filed this claim. It is mandatory for you to enter comments in this field in case you are sending the claim back to the dealer for further information or if you are denying the claim.

Actions

The following actions are allowed on a claim in this folder:

  • Send for CP Review: Select this action to send the claim to CP advisor for review. Use the drop-down field following this action to select a CP advisor to whom you want to send the claim for review.

  • Put on Hold: Select this action if you want to put this claim on hold. This claim will then be moved to your In Progress folder. You can do any changes to the claim and take similar actions on it.

  • Request Information from Dealer: Select this action if you need more information from the dealer on this claim. The claim will be forwarded to the dealer who filed it and it will be moved to the Forwarded inbox of the dealer. Incase if external user, the claim would move to the 'Forwarded Externally' inbox and incase of internal users, it would move to 'Forward Internally' inbox.

  • Request for Advice: Select this action if you want to send the claim internally to an authorized OEM user for an advice. Use the drop-down field following this action to select an internal user to whom you want to forward this claim for advice. When the OEM personnel or the dealer reply to your claim, the response is shown in the 'Replies' inbox.

  • Transfer to: Select this action to transfer the claim to another processor for manual adjudication. Use the drop-down field following this action to select an authorized processor or warranty administrator to whom you want to transfer this claim. The claim will now appear in the 'Transferred' inbox of the processor to whom the claim has been transferred.

  • Deny: If you want to deny this claim completely, then select this radio button. Use the drop-down field following this action to select a rejection code stating the reason for denying this claim.

  • Accept: If you want to accept this claim, then select this radio button. Use the drop-down field following this action to select an acceptance code stating the reason for your accepting this claim.

  • Re-process: If you want to reprocess this claim, then select this radio button.

Note: Once a claim is moved from the New folder to any other folder, it will not come back to this folder.

 

You can see the following action buttons on this page:

  • Cancel: Click on this button if you want to close this page without saving any details entered by you for this claim.

  • Validate: Click on this button to run the claim through the processing rules for validation and verification. The Validate pop-up window will be shown. Click here to know more about the Validate window.

A success message will be shown confirming your action. The message will hide itself and the current page will be closed. The claim will now be routed according to the decision taken by you. If you added any new parts in the Components Replaced section, then the dealer be accordingly notified about it.

  • Manual Supplier Recovery Initiation: The Warranty Processor can manually create the Supply Recovery Claim before the original warranty claim is closed. To initiate this process, click the Manual Supplier Recovery Initiation button. This displays the Manual Supply Recovery Initiation window. Here, select the part for which you want to initiate the Recovery Claim then click Create Supply Recovery claim. This action navigates you back to the Processor page. Take your desired decision for the claim from this page and then click Validate. After validation, click Submit to submit this claim.

Note: The removed part or causal part should have an eligible supplier recovery contract to initiate manual supplier recovery. Otherwise, the following error message is displayed on initiating manual supplier recovery: "Could not find any eligible supplier contract. Cannot initiate recovery."

Thus, when a claim is sent for review, the processor receives it in his/her "New" inbox. The system displays an option "approve and transfer to next user" that indicates that the user approves the claim but wishes to transfer it to a user who has the authority to approve the claimed amount. The system then sends the claim into the next user's pending authorization inbox.
