item groups

The Warranty Management application allows you to group various items like parts and trucks and arrange the groups in a hierarchical tree structures known as schemas.

 

To view item groups page,click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Groups folder. All links under this folder are shown. Click on the Item Groups link.

You can view all currently available schemas listed in a tabular format. To know more about navigating through the table, see Lists.

 

You can either create a new schema or add/remove groups from existing schemas.  

 

Creating Schemas

To create item groups, it is first necessary to create a schema. You can then create various item groups, at different levels in this schema.

 

To create a schema,

  • Click on the Create Schema button. The New Item Schema page is shown.

  • In the Name field, enter a name with which you wish to save this schema.

  • In the Purpose selection box, you can view all purposes that are currently not attached to any schema. Select a purpose for your schema.

  • Click Cancel to exit from this page without saving any details entered by you.

  • Click Save to save this schema. The schema is saved and the Manage Member Groups button is shown on the New Item Schema page.

Note: You can attach any number of purposes to a schema. but a purpose can be attached to one schema only.

 

Creating Item Groups

You can create item groups at various levels inside a schema.

 

To create an item group,

  • Click on the Manage Member Groups button on the New Item Schema page. The Manage Item Groups for <schema name> page is shown.

  • Click on the Create Item Group button. The New Item Group page is shown.

  • In the Group Name field, enter a name with which you wish to save this group.

  • In the Group Description field, enter a short description for the group.

  • Click Cancel to discard the new group and exit this page.

  • To create a group consisting of other item groups, select the Consisting of Groups radio button.

  • To create a group consisting of items, select the Consisting of Items radio button.

  • Click Add Items/Add Groups button. A pop-up window is shown. Enter the following details in this window:

  • In the Item Number/Group Number field, enter the item numbers of items or the group name of the groups you wish to add to your group.

  • In the Description field, enter the description of the items/groups which you wish to add to your group.

  • Click Search. All items/groups with the given parameters will be listed.

  • Select the check-box against the items/groups that you wish to add to your group.

  • Click Add Items/Add Groups button. The selected items are added to your group.

  • Click on Create Item Group button. The group with the selected items/groups is created.

You can add new items or item groups to existing groups or remove items or item groups from them.
