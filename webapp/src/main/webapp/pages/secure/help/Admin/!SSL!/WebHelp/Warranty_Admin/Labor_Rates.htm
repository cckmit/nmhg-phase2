Labor Rates

Whenever an truck fails, the Dealer must carry out some repair/replacement job in order to get the truck back in working condition. The Warranty Management application allows you to predefine the time allotted for each act of labor performed on the failed truck.

 

The Warranty Management application you to define labor rate rules based on the following parameters: dealer name/group, type of claim, warranty type, product and customer. You can set a rate for labor performed for each unique combination of these parameters. Labor rates are set up for a dealer before the dealer can file a warranty claim. These rates are applied to the servicing details that dealer files for the claim. You can create, search and maintain labor rate configurations in an easy and efficient manner.

 

To view the Labor Rates page,

  • Click on the Labor Rates link under the Common Actions column on your Home page.

  • Click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Flat Rates folder. All links under this folder are listed. Click on the Labor Rates link.

Either of the above actions will take you to the Labor Rates page. In this page, you can see the Labor Rates table that lists all configurations created by you. See Lists for more information on navigating through the table.

 

The table in the Labor Rates page provides the following information about each configuration:

  • Dealer/Dealer Group: Name of the dealer or the dealer group to which this configuration is applicable.

  • Products: Products to which this configuration is applicable.

  • Warranty Type: Warranty type to which this configuration is applicable.

  • Claim Type: Type of claim to which this configuration is applicable.

  • Customer: The customer to which this configuration is applicable.

You can view/edit an existing configuration or create a new configuration from the Labor Rates page.

Creating a Labor Rate

  • Click on the option.

  • The Labor Rate Configuration page is displayed. This page is divided into three sections:

  • Conditions

  • Rates

  • Labor Rate Audit History

In the Conditions section, specify the following details:

  • Dealer: Enter the name of the dealer to whom the new rates are applicable. You can associate the new rate with a group of dealers by clicking on the Specify Dealer Group link. The Dealer field now changes to Dealer Group field and you can enter a group name here. To switch to the Dealer field, click on the Specify Dealer link. This field supports auto-complete.

  • Product: Name of the product to which the new rates are applicable. This field supports auto-complete.

  • Warranty Type: Use this drop down option to specify the type of warranty to which the rates are applicable. The options available in this field are: All, Standard, Extended and Policy. You an select a particular warranty type or select All to apply the rates to all warranty types.

  • Claim Type: Use this drop down option to specify various claim types to which the new rates are applicable. The options available in this field are: All, Machine, Parts and Field Product Improvement.  You can select a particular claim type or select All to apply the rates to all claim types.

  • Customer Type: Use this drop down option to specify various customer types to which the new rates are applicable. The options available in this field are: All, Dealer, Direct Customer and End Customer.  You can select a particular customer type or select All to apply the rates to all customer types.

  • Comments: Enter your comments in this section.

In the Rates section, you can specify rules to apply labor rates for specific repair date ranges:

  • Click on icon. This adds a row allowing you to specify the repair date range and the rate.

  • Repair Date: Use the option in the From and To fields to specify the potential date range. The date will be displayed in the format selected by you.

  • Labor Rate: Specify the applicable rate in this field.

  • You can add more rows and set more rules for different date ranges.

  • To delete a rule, click on icon corresponding to that row.

The Labor Rate Audit History section enables you to see the Audit history of already existing labor rates. This section displays the following details:

  • Dealer/Dealer Group: Displays the dealer names or dealer group names who modified this labor rate.

  • Last Modified Date: Displays the date and time when this labor rate was last modified.

  • Modified By: Displays the name of the administrator who modified this labor rate.

  • Comments: Displays the comments, if any.

  • Status: Displays the status as inactive or active.

  • Details: Displays a link View for an inactive labor rate.

Click Cancel to close this page without saving any data entered by you.

Click Save to save the new rate. A success message is shown indicating the new configuration is saved.

Note: When creating rules for a configuration, you need to conform to the following:

  • To date cannot be before the From date.

  • From date can be a value placed in past, present or future.

  • To date should be a value placed in future.

  • If more than one rules are created, the date difference in the To date of one rule and the From date of the consecutive rule should be a single day.

  • You cannot assign overlapping values to From and To date fields.

 

Editing a Labor Rate

You can edit a labor rate configuration and reset the parameters to suit your needs.

 

To edit a configuration:

  • Double click on a configuration from the Labor Rates page to open it in the detailed page.

  • You can edit any field in this page.

  • Click Cancel to close this page without saving any changes made by you.

  • Click Save to save the edited rate. A success message will be shown indicating all changes are saved.

  • To delete this configuration, click Delete. A success message will be shown indicating the configuration is deleted.

Note: You can edit the To date to a value in the past

Viewing Labor Rate Audit History

 

To view the history of an inactive labor rate:

  • Click on the link View under Details in the Labor Rate Audit History section.

  • This displays the complete history for that labor rate.
