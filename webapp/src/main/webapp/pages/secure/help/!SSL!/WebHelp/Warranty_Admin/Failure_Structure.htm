Failure Structure

Failure hierarchy streamlines the process of identifying the failure in an equipment. Established as a tree structure, it helps to identify the exact location of failure, kind of failure, cause of failure, job code for the labor required to repair the failure. The warranty administrator can setup a virtual structure to indicate the assemblies, sub-assemblies and parts which make up a product.

 

Click on the Manage Failure Structure folder. You will find the following links in this sub-section:

  • Product Failure Hierarchy

  • List Fault Locations

  • List Service Procedures

  • Maintain Failure URL of Failure Detail

Click on a link to view the details under it. A new page, displaying the contents of the selected link, will appear in the right-hand pane.
