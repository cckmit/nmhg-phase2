COST CATEGORY CONFIGURATION

The Cost Category Configuration is used to Configure the cost category based on particular product. Dealer can file a Claim based on the configuration by an Administrator. The cost categories are displayed on claim page 2.

 

To configure Cost Category, perform the following -

  1. Click Cost Category Configuration under Warranty Admin in the left hand pane. This displays the list of products covered under the claim filing screen.

  2. Select or de-select the Cost Category or Products Covered and click Update. After updating, the application displays a Successful message "Cost Categories updated Successfully".
