MANAGE WAREHOUSES

Certain parts marked for return to the OEM must be returned to specific locations. These locations are referred to as warehouses. The Warranty Management application allows you to view, create and maintain warehouses in an easy and efficient manner.

 

You can define the following data for each warehouse: warehouse name, its location address, city, state, country. You can assign a receiver, inspector and a part shipper to each warehouse. The application also allows you to create bins for each warehouse.

 

To view the warehouse folder, click on the Warranty Admin tab on your left-hand pane. All folders under this tab are shown. Click on the Manage Warehouses folder. All warehouses created by you will be listed in the Warehouses table on this page. To know more about navigating through the Warehouses table, see Lists.

 

The Warehouses table provides the following information about each warehouse:

  • Warehouse Name: Name of the warehouse.

  • Address: Complete postal address of the warehouse.

  • City: Name of the city where the warehouse is located.

  • State: Name of the state where the warehouse is located.

  • Country: Country where the warehouse is located.

To view details of a warehouse, click on it in the table. You can view all details of a warehouse on a single page by double clicking on it in the table. The <warehouse name> page is shown. The warehouse page is divided into various sections for ease of navigation. The following sections are shown:

  • Details: Displays address details of the warehouse.

  • Users: Displays information about personnel associated with the warehouse.

  • Bins Info: Displays details of all bins existing in the warehouse.

You can create a new warehouse or edit details of an existing warehouse, on the Manage Warehouses page.

 

Creating a Warehouse Configuration

You can create a new warehouse by specifying a name and location for the warehouse.

 

To create a new configuration:

  • Click on the option.

  • You will see the New Warehouse page.

In the Details section, specify the following details:

  • Warehouse Name: Name of the warehouse.

  • Address: Complete postal address of the place where the warehouse is located.

  • City: Name of the city where the warehouse is located.

  • State: Name of the state where the warehouse is located.

  • Country: Name of the country where the warehouse is located.

In the Users section, you can assign personnel to the new warehouse:

  • Receiver: Personnel responsible for collecting the returned parts and assort them into different bins. All available receivers are listed in this column. You can assign a receiver to the warehouse by selecting the radio button alongside that receiver.

  • Inspector: Personnel responsible for inspecting the returned parts for proposed defects. All available inspectors are listed in this column. You can assign multiple inspectors to the warehouse by selecting the radio button alongside that inspector.

  • Part Shipper: Personnel responsible for shipping the defective parts to respective suppliers for replacement or reimbursement. All available part shippers are listed in this column. You can assign a part shipper to the warehouse by selecting the radio button alongside that part shipper.

Bins are the locations in the warehouse where the parts will be kept. In the Bins section, you can configure multiple aisles for various parts.

  • Click on icon. This adds a row allowing you to specify a bin name.

  • Enter a name for the bin in the text field provided.

  • You can add any number of bins to this warehouse.

  • To delete a bin, click on icon corresponding to that bin.

  • If you want to close this page without saving any data entered by you, click Cancel.

  • Click Save. A success message is shown indicating the new warehouse is saved.

Note: As you assign personnel to a warehouse, their names will be removed from the available list.

Editing a Warehouse Configuration

You can edit the details of an existing warehouse and reset the parameters to suit your needs.

 

To edit a warehouse:

  • Double click on a warehouse to open it in a <warehouse name> page.

  • You can edit any field in this page.

If you want to save all changes made by you, click Submit. To delete a bin, click on icon corresponding to that bin.

A success message is shown confirming your action.
