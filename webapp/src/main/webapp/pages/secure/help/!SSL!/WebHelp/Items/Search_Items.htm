SEARCH ITEM

The Warranty Management application allows you to search for a machine or part, using the Item option.

 

To go to the search page,

  • Click on the Search Item link under Go To column on your Home page.

  • Click on the Items tab on the left-hand pane. The tab will expand to show all folders under it. Click on the Define Search Query link.

Either of the above actions takes you to the Define Search Query page. You can search for items based on various item related parameters on this page. You can also save a search for future viewing. To know more about creating or editing existing business conditions, click here.

 

Search Results

The results of your search will be shown in a tabular format. Click here to know more about the listing toolbar on the Search Results table. Click here to know more about navigating through the Search Results table.

 

Edit Search

You can edit a search phrase and modify any of the search parameters.

 

To edit a search phrase,

  • Click on the button on the search result page.

  • The Edit Search Query page is shown. You can see the search conditions created by you here.

  • You can edit any parameter in this field.

  • To reset the data in this page, click Reset. All details entered by you will be reset to the default selections.

  • To execute the modified search conditions, click Execute. The search results will be shown in a tabular format.

You can also search an item by its type, Machine or Part. To view an item with the Machine item type, click on MACHINE Number Search link under the Items tab on the left-hand pane. It will list all the items with the item type as Machine. Similarly, to view an item with the Part item type, click on PART Number Search link under the Items tab on the left-hand pane. Double click on an item to view the item details.
