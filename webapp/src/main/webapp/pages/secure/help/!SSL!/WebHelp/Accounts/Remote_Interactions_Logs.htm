REMOTE INTERACTIONS LOGS

When you upload a sync, sometimes the sync gets completed and sometimes it fails. This feature enables a user to see what entries failed in the sync.

 

Perform the following steps to view the Remote Interaction Logs:

  • Click on Remote Interaction Logs under the Accounts tab under the Warranty Admin tab on the left-hand pane.

  • Select an interaction name from the Interaction Name dropdown such as Claim or CreditNotification.

  • Select the desired status Failed or Completed.

  • Enter values in other fields if desired since these are optional and then click Search.

  • Search in Hidden Records: Select this checkbox to search for the sync in the hidden records. You can do this to increase the volume of records.

  • Clicking Reprocess will reprocess the records again.
