ON HOLD

If you mark a claim to be put on hold, from any of the claim folders, then the claim will be moved to your On Hold folder. You can do any changes to the claim as in the Processor Review folder and take similar actions on it.

 

To view all on hold claims, click on the Claims tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the On Hold folder. On this page you can see the On Hold table that lists all claims marked for hold by you. For further information on navigating through the table, see Lists.

 

You can see the following details for each claim in the On Hold table:

  • Claim Number: Reference number of the claim.

  • Claim Type: Kind of claim. A claim can be of type machine, parts or campaign.

  • Dealer: Name of the service provider who filed this claim.

  • Fault Code: Failure code associated with this claim.

  • Causal Part: Part serial number or part number of the part which caused the failure in the equipment.

  • Failure Date: Date when the equipment failed.

  • Repair Date: Date when the equipment was repaired.

You can view the details of any claim by clicking on it. Double clicking on a claim will open it in a new page, <claim number>. The claim detail page for a claim in this folder is divided into various sections for ease of navigation: Processing Errors, Notifications, Claim History, Claim Header, Equipment Details, Failure Details, Service Details, Components Replaced, Miscellaneous, Supporting Documents, Payment Details and Actions. The fields under some sections on this page may be hidden. To view the fields, click on the section header. This will expand the section showing all fields under it. Likewise, clicking on the section header of an expanded section collapses it hiding all fields under it.

 

For further information on the data shown in claim detail page for a claim in the On Hold folder, click here.

 

Actions

You need to enter comments in the following fields for a claim in the On Hold folder:

  • Internal Comments: The comments that you enter in this field are visible only to other OEM users.

  • Comments to Dealer: The comments that you enter in this field are visible to both the OEM users as well as to the dealer who filed this claim. It is mandatory for you to enter comments in this field in case you are sending the claim back to the dealer for further information or if you are denying the claim.

The following actions are allowed on a claim in this folder:

  • Put on Hold: Select this action if you want to keep this claim in the On Hold folder.

  • Request for Information: Select this action if you need more information from the dealer on this claim. The claim will be forwarded to the dealer who filed it and it will be moved to your Forwarded inbox. Click here to know more about the Forwarded claims.

  • Request for Advice: Select this action if you want to send the claim internally to an authorized OEM user for an advice. Use the drop-down field following this action to select an internal user to whom you want to forward this claim for advice. Click here to know more about Request for Advice.

  • Transfer: Select this action to transfer the claim to another processor for manual adjudication. Use the drop-down field following this action to select an authorized processor or warranty administrator to whom you want to transfer this claim. Click here to know more about Transferred claims.

  • Accept: If you want to accept this claim, then select this radio button. Use the drop-down field following this action to select an acceptance code stating the reason for your accepting this claim.

  • Deny: If you want to deny this claim completely, then select this radio button. Use the drop-down field following this action to select a rejection code stating the reason for denying this claim.

You can see the following action buttons on this page:

  • Cancel: Click on this button if you want to close this page without saving any details entered by you for this claim.

  • Validate: Click on this button to run the claim through the processing rules for validation and verification. The Validate pop-up window will be shown. Click here to know more about the Validate window.

A success message will be shown confirming your action. The message will hide itself and the current page will be closed. The claims will now be routed according to the decision taken by you. If you added any new parts in the Components Replaced section, then the dealer be accordingly notified about it.
