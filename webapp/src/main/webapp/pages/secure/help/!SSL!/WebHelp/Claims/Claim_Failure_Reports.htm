CLAIM FAILURE REPORTS

The claims failure reports page lists all the claims which have the failure reports pending on them.

When a claimant is submitting a claim and clicks on Validate the system checks if a warranty failure analysis report is required to be submitted. If it is required, then the system displays a message to the user indicating the submission of the failure report. When the user hits the Continue to submit failure reports button the system saves all the details of the claim including payment information and takes the user to the failure report page where you can perform any of the following actions:

  • Save: Saves the report and sends the claim to the Claims Failure Reports inbox. When the user opens the report next to be filled out all the saved details are populated again.

  • Cancel: Closes the tab without saving any details.

  • Submit: Submits the report.

Once the report is submitted, the claim is also submitted and assigned a claim number.

You can view the claim failure reports by clicking on Claim Failure Reports link under the Claims tab on the left pane.
