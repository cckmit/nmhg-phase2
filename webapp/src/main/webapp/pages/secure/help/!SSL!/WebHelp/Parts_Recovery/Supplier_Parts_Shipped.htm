SUPPLIER PARTS SHIPPED

When you ship a part or a group of parts to a location specified by the OEM, an entry for that shipment is shown in the Supplier Parts Shipped folder for your reference. These entries will remain in this folder till it is received at the shipped location. All details shown in this folder are read-only.

 

To view the shipment details, click on the Parts Recovery tab on the left-hand pane. The tab will open to reveal the folders under it. Click on the Supplier Parts Shipped folder. On this page you can see the Shipped Parts table that lists all shipments made by you. For further information on navigating through the table, see Lists.

 

The Shipped Parts table provides the following information about each consignment:

  • Shipment Number: Identification number for the consignment. This number is generated when you click Generate Shipment for one or more due parts.

  • Tracking Number: Tracking number for your shipment provided by the shipper.

  • Carrier: Name of the shipper you used for shipping the parts to the return location.

  • Return Location: Name of the location to which you are making the shipment.

  • Shipment Date: Date when you shipped the parts.

You can view the details of any shipment by clicking on it. Double clicking on a shipment will open it in a new page, <location name>. The detail page is divided into three sections, Claim Details, Part Return Details, and Shipment Details for ease of navigation. The Claim Details section displays information about the claims to which the parts belongs. The Part Return Details will show all the parts that you shipped in this shipment. The Shipment Details section will show information about the shipment.

 

You can see the following details in the Claim Details section:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

  • Serial Number: If the claim was a machine claim, this field will display the serial number of the machine. If it was a campaign claim, this field will show the serial number of the machine on which you filed the campaign claim.

  • Part Serial Number: If the claim was a parts claim, then you can see the part serial number of the part on which the claim was filed, in this field. If the part is non-serialized, then you can see this field as blank.

  • Item Number: If the claim was a machine claim, this field will show the item number of the machine.

  • Part Number: If the claim was a parts claim, this field will show the part number of the part on which the claim was filed.

  • Model: This field will show the model of the machine or the part depending on the type of the claim.

In the Part Return Details section, the following information is displayed in the preview pane and the detail page:

  • Serial Number: Serial number of the part that is to be returned. If the part is non-serialized, then this filed will appear blank.

  • Supplier Part Number: Part number of the part, as identified by the supplier.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • Quantity: Number of parts of the given specification that are to be returned. If the part is serialized this value will always be 1.

If you have included parts belonging to different claims in this shipment, then you will see the above two sections repeated for each unique claim.

 

You can see the following details in the Shipment Details section:

  • Carrier: Name and debit account number, if any, of the carrier that you used to ship the parts.

  • Shipment Date: Date when you shipped the consignment.

  • Tracking Number: The unique tracking number provided by the shipper for your shipment. This number allows you to track your shipment to its return location.

  • Freight Charges: If you had entered any freight expenses, then you can see the same in this field.

  • Comments: If you had added any additional comments about the shipment, parts being shipped, carrier used for the shipment, then you can see the same in this field.
