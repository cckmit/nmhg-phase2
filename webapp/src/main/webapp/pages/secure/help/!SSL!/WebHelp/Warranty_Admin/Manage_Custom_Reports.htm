Manage Custom reports

Administrator can set up the the below custom reports :

  1. Arrival Condition Report (ACR) :- The Arrival Condition Report (ACR) records the machine information at the time of arrival at the Dealer's premises.

  2. Pre-Delivery Inspection (PDI) :- The Pre-Delivery Inspection (PDI) Report records the machine inspection information before inventory delivery is made by the Dealer to the End Customer.

Creating a Custom Report

Navigate to Manage Custom reports link under Warranty Admin and click on Create custom report button.

  • Enter the report name.

  • Select the custom report type (ACR, PDI, Failure).

  • Select Inventory Type (Stock or Retail). Present as is in the system.

  • The Administrator should be able to select one or more product types. To select product type, click Add Row and enter the product details.

  • After entering the Product details, enter the applicable parts by specifying the part details in the Applicable Parts section. Select the type of Applicability.

  • The Sections section is for questions or description of the product. For more information on adding question, click here.

  • Click Save to save the report.

  • Click Publish to publish the report on the Dealer's page.

  • Click View All Questions to view the questions.

Note:

  • When we click save, the report gets saved. We can add questions to existing custom report as well as edit it.

  • When we click publish then we cannot modify question and Add Question Link disappears. To Publish we need to add at least one question in section of the report.

  • After Publishing the report, the Admin can modify the below details.

  • Report Name

  • Report Type

  • Add/Remove Product Type

  • Add/Remove Applicable Parts
