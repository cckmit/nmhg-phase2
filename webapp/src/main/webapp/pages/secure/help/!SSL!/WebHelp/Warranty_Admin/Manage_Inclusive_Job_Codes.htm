MANAGE INCLUSIVE JOB CODES

A service procedure is identified using the job-code associated with it. There are few job codes that are related and cannot be selected. For instance, if you have selected replaced as the job code, you cannot select the addition of a part as a job code. The admin has the right to put a restriction to the use of job codes that are associated.

 

To create inclusive job codes,

  • Click on Manage Inclusive Job Codes under the Warranty Admin tab on the left-hand pane. You can view the list of parent job codes along with their description. Double click on the parent job code to see the inclusive job codes.

  • Click on Create Inclusive Job Code button in the listing toolbar.

  • Enter the parent job code for which you want to create inclusive job codes.

  • Click on the button to add a job code row

  • Enter the job code which you want to be included under the parent job code.

  • Click Save.

You can add more than one job code under the parent job code by clicking on the button. To delete a row, click on .

You can also update the inclusive job code list by double clicking on the parent job code listed on the Manage Inclusive Job Codes page. You can add/delete a job code and click Save to update the changes made.
