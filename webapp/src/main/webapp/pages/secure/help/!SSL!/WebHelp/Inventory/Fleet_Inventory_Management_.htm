FLEET INVENTORY MANAGEMENT

The Fleet Inventory Management section contains all the install base services such as Dealer to Dealer Transfer, Equipment Transfer, Retail Machine Transfer, etc. that administrator can perform on an inventory. You can see the following folders here:

  • Dealer to Dealer Transfer

  • Warranty Registration

  • Equipment Transfer

  • Retail Machine Transfer

  • Scrap/Unscrap Inventory

  • Manage Warranty Coverage

  • GoodWill Warranty Extension

  • Extended Warranty Purchase

Click on a folder to view the links associated with them.
