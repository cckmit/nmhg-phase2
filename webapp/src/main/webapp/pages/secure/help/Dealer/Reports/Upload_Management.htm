UPLOAD MANAGEMENT

The Upload feature allows you to perform any operation on a number of records. For example, if you want to file a Delivery Report on a bulk of serial numbers, then instead of filing the Delivery Report for each inventory one by one, you can upload all the details in a template and upload the template for all inventories together.

 

Perform the following steps for Upload Management:

  • Click on Manage Uploads/Downloads tab under the Reports tab on the left-hand pane.

  • Click Upload Management. The Upload Management page appears. This page contains several links such as Install Base Upload, Delivery Reports and so on.

  • Click on the desired link to download its template file to a location in your system . For e.g. click on the Delivery Reports link to download the DeliveryReportUpload template file.

  • Enter the required details in this file and save it to your system.

  • Select the type of upload from the Select type of upload drop-down, locate the file from your system by clicking Browse and then click Upload.

To check the status of the uploaded file, click View Upload History for a particular type.  The Upload History page appears which displays the details of all the uploaded records such as File Name, status of upload and other information.

Double click on a record, to view more specific details of that record. This page also provides you an option to download the error file or download the template file which you have uploaded.

Note: All the Delivery Reports and claims go into draft status and you should enter the required information and submit the corresponding claims and Delivery Reports.
