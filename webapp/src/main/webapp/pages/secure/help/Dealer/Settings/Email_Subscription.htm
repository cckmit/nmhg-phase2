EMAIL SUBSCRIPTION

The Warranty System has the feature to send email alerts to the users. Currently this system has different users like Admin, Dealer, Processor, SRA and Supplier. The system displays different email subscription events based on the user roles. If the user does not want all email alerts, then the user has the choice to select the events from the list of Email Subscription Events which are displayed for their roles.

 

To view and select email subscription events, click on Email Subscription tab under the Settings tab on the left-hand pane. The Dealer contains the following events under Email Subscription Events Section:

  • Claim Accepted And Closed

  • Waiting for Part Return

  • Claim Denied For Lack Of Parts

  • Claim Denied And Closed

  • Claim Forwarded

  • Waiting For Labor FOC Claims

  • Part Moved To Overdue

  • Part Received

  • Claim Re-accepted

  • Service Manager Response

  • Part To Be Returned

Once the user selects the event for which email notification is required, click Update.
