USER MANAGEMENT

A dealership contains many users, some of them can be service technicians who handle repairing of failed inventory or salesmen who sell inventory items to end customers. The User Management folder in the Settings tab allows you to manage the users associated with your dealership.

 

To create a user perform the following,

  • Click on User Management tab under the Settings tab on the left-hand pane. Click on Create User.

  • Select the Company Name or specify the Company Number under which you want to create a user.

  • Assign a role to the user by selecting a role from the available list of roles displayed under Available Fields. You can select multiple roles by pressing the Ctrl key and simultaneously clicking on the roles.

  • Click on   to move the selected roles under Selected Fields. To remove a role from the Selected Fields column, select the particular role and click on .

  • Enter the First Name and the Last Name.

  • Select the Locale from the drop-down list.

  • Enter the Address Line of the user.

  • Select Country from the drop-down list and enter name of the State, City, Zip/Posteal Code, Phone, E-Mail ID and Fax number of the user.

  • Provide a Login ID, Password and Confirm the Password.

  • Click Add User. This creates a new user.

The Warranty administrator has the rights to update an existing User.

 

To update a user perform the following,

  • Click on User Management tab under the Settings tab on the left-hand pane. Click on Update User.

  • Enter the Login ID of the user you want to update details of and click Search User.

  • Update the user details. You can also update the role assigned to the user.

  • Enter the Password and confirm the password.

  • Select Activate to update the user. If you wish to delete the user, select DeActivate.

After you make necessary changes to the user profiles, you can do one of the following actions:

  • Cancel: If you want to close this page without saving any details entered by you, then click on this button.

  • Update User: Click on this button to save all changes made by you. A success message will be shown confirming your action.
