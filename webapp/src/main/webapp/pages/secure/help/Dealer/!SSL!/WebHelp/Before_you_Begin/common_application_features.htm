Common Application Features

This section details some features commonly seen across the Warranty Management application. The descriptions provided here hold good for all instances where these features appear in the application.

 

list

When a folder or an option in a folder is selected on the left-hand pane, a corresponding tab is shown in the right-hand pane. Some of these tabs have a tabular structure that lists related items. The columns in this tabular structure vary with each tab and are specific to the selected folder.

 

Some features common to all lists:

  • The first column of every list is a column of check-boxes. Whenever a row is selected, the check-box corresponding to that row will be automatically selected. You can also select a row by checking the check-box corresponding to that row. You can select multiple check boxes at a time.

  • By default, the list shows 7 entries. The first entry in a list is selected by default and all details of this entry are shown in the preview pane.

  • You can hide the preview pane by clicking on the icon. The listing table now displays 25 entries. To view the preview pane, click on icon.

  • A Navigation Toolbar is shown at the rightmost end of a list,

  • All entries in a list are paginated and the first page is shown by default. Observe the Page m of n display on the Navigation Toolbar. Here m is the current page and n is the total number of pages in the list.

  • At any point links to only 5 pages are visible in the Navigation Toolbar. Also, at any point, the link to the current page along with the links to two preceding and two succeeding pages are shown.

For example: If there are 30 pages in the list and you are currently on page 1, then links to pages 1, 2, 3, 4, 5 will be displayed. If you are currently on page 15, then links to pages 13, 14, 15, 16, 17 will be displayed and if you are currently on page 29, then links to pages 26, 27, 28, 29, 30 will be displayed.

  • Clicking on the link of any page will take you to that particular page.

  • Use the forward and backward arrows to move between the pages.

  • Clicking on the icon takes you to the first page, i.e., page 1.

  • Clicking on the icon takes you to the last page, i.e., page n.

  • Clicking on the icon takes you to the page preceding the current page, i.e., if you are on page 4, clicking this icon will take you to page 3.

  • Clicking on the icon takes you to the page succeeding the current page, i.e., if you are on page 4, clicking this icon will take you to page 5.

  • Use the drop down option to directly navigate to a particular page. This drop down option lists all pages available in the list.

  • You can sort the entries in a list in the ascending or descending order of its columns. You can also perform multiple sorting on the list. See Sorting a List for further information.

  • You can perform a "begins with" filtering on the entries of a list. See Filtering a List for further information.

  • A Listing Toolbar is shown on the top right corner of a list, . See Download to Excel, Print, Refresh and Delete for further information.

Preview Pane

The preview pane shows details relevant to the entry selected in the list. Preview pane can be of one of the following type: plain or tabbed, depending on the details available.

  • A plain preview pane displays all details of the selected entry and allows you to edit each field and update the entry.

  • If the details are large to be fit in the plain preview pane, a tabbed preview pane is shown. Each tab in a tabbed preview pane represents a different section of the selected entry.

  • Save and Cancel buttons are available on the preview pane.

Sorting a List

You can sort a list in the ascending or descending order of its columns. By default, the entries in a list are sorted in the ascending order of the first column, unless specified otherwise. The ascending sort icon, is shown in the first column header, indicating the current sorting.

  • You can sort the entries, column-wise, in ascending or descending order, by clicking on a column header.

  • If a column has not been sorted before, by default, one click sorts a column in the ascending order.

  • When a column is sorted in the ascending order, the ascending sort icon, is shown in the column header.

  • When a column is sorted in the descending order, the descending sort icon, is shown in the column header.

  • A click on an already sorted column will sort it in the reverse order.

  • You can perform multi-level sorting by pressing the Ctrl key and clicking on different columns.

Filtering a List

You can perform a "begins with" filtering on the entries in a list. Filtering is allowed on each column of the list.

  • To filter a list, you need to enter relevant text in the filter fields provided just below the column header in each column.

  • As you enter text in a filter field, the list is filtered to show only those entries that begin with the specified filter word.

For example: If you type “CA” in the Serial Number filter, the list will filter the entries and start showing only those serial numbers which start with “CA”.

  • As a list is filtered, the pagination changes to reflect the new filter results.

  • If you had previously applied a sorting to the list, then the results of the filtering will be sorted in the previously defined manner.

  • You can perform multilevel filtering by entering text in various filter fields in the list.

Download to Excel

You can download entries on the current page of a list or all entries in a list into an Excel sheet using the Download to Excel option.

  • To download an entries to an Excel sheet, make sure you are on the page which you wish to download.

  • Click on option. A pop-up window is shown.

  • To download current page, select the Download Current Page radio button.

  • To download all entries of a list, select Download All Results radio button. You can download a maximum of 500 entries using this option.

  • Click Download. The selected entries will be downloaded to an Excel sheet. The File Download dialog box is shown which allows you to save the Excel sheet to your desired location.

The columns in the Excel sheet will be same as the columns visible in the listing and in the same order. Also the downloaded entries will be filtered and sorted the same way as the listing was at the time of downloading.

 

Print

The Warranty Management application allows you to print a selected entry and also print all entries in a list. The Print option is provided in the Listing Toolbar.

 

To print a page:

  • Select the entry that you want to print by clicking on it.

  • Click on the option.

  • A pop-up window is shown displaying a printer-friendly version of the selected entry.

  • The print dialogue box appears and you can set the printing attributes in this dialog box.

  • Click Print to print the page.

Refresh

The Refresh option, provided in the Listing Toolbar allows you to refresh the list as well as the preview pane. Refreshing the list will return the list to its default settings, i.e., any filtering and sorting action performed on the list will be lost.

 

Delete

The Delete option is provided in the Listing Toolbar. This option allows you to selectively delete entries from the listing.

 

To delete an entry:

  • Select the entry that you want to delete by clicking on it.

  • Click the option. The selected entry is deleted from the list.

  • You can delete multiple entries from the list by pressing Ctrl key and clicking on the entries. After selecting the relevant entries, click Delete.

  • In some lists, the Delete option maybe disabled.
