prepare Due Parts

After processor takes decision on the claim and if the claim is not rejected, then parts are added to Dealer’s Prepare Due Parts inbox. This indicates the dealer to collect the parts which may ask for return by the Processor on the scheduled date or even on an ad-hoc basis.

To view the parts in the Prepare Due Parts inbox, click on the Part Returns tab on the left-hand pane. The tab will open to display the folders under it. Click on the Prepare Due Parts folder.

Views

Two types of views are available for you to view the data in the Prepare Due Parts table: Location View and Claim View.

 

Location View

The location view consolidates all the parts across the claims that are to be shipped to the same location. To view the parts in this view, click the Location View button. You can see all parts grouped by return locations,  in a tabular format.

 

The Location View table provides the following details about each return location:

  • Return Location: Name of the return location.

  • Location Number: Location Number of the return location.

Double clicking on a return location will open the claims to be returned to this location. The detail page is divided into two sections, Claim Details and Part Return Details: The Claim Details section displays information about the claim to which the part belongs. The Part Return Details displays information about all the parts that are to be returned to this location for the same claim.

 

The following details are displayed in the Claim Details section::

  • Claim Number: The number of the claim to which the part to be returned belongs. The Claim Number is a hyperlink to the Claim Detail page. Click here for more information on the Claim Detail  page.

  • Serial Number: This field displays the serial number of the machine. The Serial Number is a hyperlink to the Truck Detail page. Click here for more information on the Truck Detail page.

  • Model Number: This field will show the model of the machine on which the claim was filed.

In the Part Return Details section, the following information is displayed:

  • Select Checkbox: If you want to include this part in your shipment, then select this checkbox.

  • Part Number: Part number of the part that is to be returned.

  • Description: A short description of the part that is to be returned.

  • To be Shipped: Number of parts of the given specification that are to be returned.

  • RMA Number: The RMA Number of the part to be returned.

  • Dealer Name: The name of the dealer for this part return.

Claim View

The claim view consolidates all parts belonging to one claim, that must be returned to the OEM. The parts may have different return locations. To view the parts in this view, select the Claim View option in the View drop-down. You can see all parts belonging to individual claims, grouped together in a tabular format.

 

The Claim View table provides the following details about each claim:

  • Claim Number: The number of the claim to which the part to be returned belongs. The claim number is a hyperlink to the claim detail page. Click here for more information on the claim detail page.

Double clicking on a claim will open it in a new page, <claim number>. The detail page is divided into two sections, Claim Details and Part Return Details. These sections are discussed in the Location View above.  

Actions

Once in a month, the WPRA is generated for the entire returnable parts and Dealers are notified in their Due Parts inbox. The items which are moved to Due Parts inbox are thus removed from the Prepare Due Parts inbox of Dealer.
