CLAIMED RETURNED PARTS

When the Part shipper generates the shipment and prints the shipment tag, the dealer receives the shipping parts details in his Claimed Returned Parts inbox.  This inbox displays the complete list of returned parts. The details page is read only and is only for history purpose.

 

To view all the parts in the Claimed Returned Parts inbox,

  • Click on the Claimed Returned Parts under the Parts Return tab on the left hand pane, or

  • Click on the Claimed Returned Parts link under the Action Folders on the Home page.

Either of the above options will take you to the Claimed Returned Parts page. This page displays the following details:

  • Shipment Number: The shipment number of the part to be returned.

  • Return From Location: The return location of the part.

  • Carrier: The carrier for the part.

  • Tracking Number: The tracking number of the returned part.

  • Shipment Date: The shipment date of the returned part.
