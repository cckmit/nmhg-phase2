DUE PARTS INSPECTION

The Due Parts Inspection folder contains details of all parts forwarded by the receiver for further inspection. The data in this folder allows you to efficiently manage all parts marked for inspection.

 

To view all parts marked for inspection, click on the Part Returns tab on the left-hand pane. Click on the Due Parts Inspection folder. On this page you can see the Parts for Inspection table that lists all parts received in this warehouse. For further information on navigating through the table, see Lists.

 

If you double click on any entry in this table, all details relevant to that part will be displayed on a new page, Claim Number <n>. You can view the following details in the Claim Number <n> page:

 

Claim Details

  • Claim Number: Claim number of the claim associated with this part.

  • Truck Serial Number: Serial number of the truck.

  • Model Number: Model number of the truck.

Part Return Details

  • Part Number: Part number of the part.

  • Location: Location where the dealer is expected to ship this part.

  • Pending Shipment: Quantity of pending shipment.

  • Shipped: Quantity of the shipped parts.

  • Cannot Ship: Quantity of the parts that cannot be shipped.

  • Received: Quantity of parts received.

  • Inspected: Quantity of parts inspected.

  • Total: The total of the parts.

  • Action Settlement Code:

  • Accept: Enter the quantity of parts accepted and select the acceptance reason from the dropdown.

  • Reject: Enter the quantity of parts rejected and select the rejection reason from the dropdown.

Manage Documents

In this section, you can attach binary documents as desired.

The following details are shown as part of this section:

  • File Name: This field displays the name of the document which you want to attach.

  • Document Type: To attach a document, select a document type from the Document Type dropdown and click on the button. A pop-up window is shown. Click on the Browse button to browse through your system for the document. After you have selected the document type and mentioned the exact location of the documents, click Attach. All selected documents will be attached to the claim. To delete an entry, click on the option.

Note: The List of documents appearing in the Document Type dropdown is a managed list of values by the Admin.

Comments

If you have any additional comments on this part, you can write the same in this text field.

 

Click Submit to submit the actions taken by you on all parts marked for inspection. A success message will be shown confirming your action. The message will disappear and the current page will be closed. The part will be removed from your Due Parts Inspection folder.
