SEARCH PARTS

The Warranty Management application allows you to search for parts sent to you for recovery by the OEM.

 

To go to the search page, click on the Supplier Recovery tab on the left-hand pane. The tab will expand to show all folders under it. Click on the Define Search Query link. You can search for equipments based on various part related parameters on this page. You can also save a search for future viewing.

 

You can search for positive results for a set of conditions that you define on this page, or you can also search for a negation of the set of conditions defined on this page.

 

 

The conditions created under one search phrase can be applied in different ways.

  • Keep the Not check-box un-checked. Select the option All of the conditions are true in the drop-down. The search results will show all replaced parts which satisfy all of your search conditions.

  • Check the Not check-box. Select the option All of the conditions are true in the drop-down. The search results will show all replaced parts which do not satisfy any one of your search conditions.

  • Keep the Not check-box un-checked. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all replaced parts which satisfy at least one of your search conditions.

  • Check the Not check-box. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all replaced parts for which do not satisfy any your search conditions.

Every search condition is made up of three parts, <attribute> <object> <value>.

The attribute is a property of the business object, i.e. the part. An operator is a mathematical operation used as part of the condition. The value part of the search condition indicates the operand against which the operator should be applied to the attribute. You will need to create a search condition using the attributes, objects and values provided in the query builder to search for claims.

 

Enter the following details to search for inventory items:

  • In the Provide a phrase describing the search condition, enter a name under which you want to save the search conditions for future searches.

  • If you want to view the results for a negation of the search conditions you create, then check the NOT check-box.

  • Select an option from the drop-down, All of the conditions are true or Atleast one of the conditions below is true.

  • In the Search Conditions section, click on the option. A new row will be added.

  • Use the drop-down to select an attribute for your search condition. For example, select Return Location from this drop-down.

  • Use the drop-down to select an object for your condition. The options in this drop-down depends on the attribute selected by you. For example, for the Return Location attribute, select contains from this drop-down.

  • Now you need to enter a value in the text field. This is a free text field. For example, enter the value Location A.

  • You can add multiple conditions for a more advanced search, by clicking on the option.

  • You can delete a condition by clicking on the corresponding to that row.

  • If you wish to save this search, then check the Save check-box.

  • If you want to clear the options selected by you, click on the Reset button.

  • If you want to search for replaced parts based on the condition you just created, click on the Execute button.

The page refreshes to show the search results in a tabular format.

 

For example,

For the condition, Return Location contains Location A, the search results will show all replaced parts whose return location contains Location A.

 

If you have selected the Save check-box, then the search will be saved under the search phrase specified by you.

 

Nested Searches

Some of the claim parameters allow you to create nested search conditions.

You can either select an object for the main attribute or add nested sub-conditions with attributes, objects and values which correspond to the main attribute.

 

For example,

For the attribute Dealer Shipment, you can create a sub-condition, Shipment Number equal 10101. The search results for such a search condition will be all replaced parts that have been shipped and whose shipment number equals 10101.

 

Search Results

The results of your search will be shown in a tabular format. Click here to know more about the listing toolbar on the Search Results table. Click here to know more about navigating through the Search Results table. To view the details on a single page, double click on that entry in the table. The page is divided into different sections, Claim Details, Equipment Details, Failure Information, Components Replaced, Part Return History, Comments, Recovery Cost Details, Supplier Recovery Credit Memo, Recovery Claim History and Credit Memo History, for ease of navigation.

 

Edit Search

You can edit a search phrase and modify any of the search parameters.

 

To edit a search phrase,

  • Click on the button on the search result page.

  • The Edit Search Query page is shown. You can see the search conditions created by you here.

  • You can edit any parameter in this field.

  • To reset the data in this page, click Reset. All details entered by you will be reset to the default selections.

  • To execute the modified search conditions, click Execute. The search results will be shown in a tabular format.

If you want to reopen a claim, click on Reopen button at the bottom of the page. Click here to know more about reopening of claims.
