SEARCH ITEMS

The Warranty Management application allows you to search for an item, using the Item option.

 

To go to the search page,

  • Click on the Search Item link under Go To column on your Home page.

  • Click on the Items tab on the left-hand pane. The tab will expand to show all folders under it. Click on the Define Search Query link.

Either of the above actions takes you to the Define Search Query page. You can search for items based on various item related parameters on this page. You can also save a search for future viewing.

 

You can search for positive results for a set of conditions that you define on this page, or you can also search for a negation of the set of conditions defined on this page.

 

 

The conditions created under one search phrase can be applied in different ways.

  • Keep the Not check-box un-checked. Select the option All of the conditions are true in the drop-down. The search results will show all inventory items which satisfy all of your search conditions.

  • Check the Not check-box. Select the option All of the conditions are true in the drop-down. The search results will show all inventory items for which do not satisfy any of your search conditions.

  • Keep the Not check-box un-checked. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all inventory items which satisfy at least one of your search conditions.

  • Check the Not check-box. Select the option Atleast one of the conditions below is true in the drop-down. The search results will show all inventory items which do not satisfy any one of your search conditions.

Every search condition is made up of three parts, <attribute> <object> <value>.

The attribute is a property of the business object, i.e. the inventory item. An object is a mathematical operation used as part of the condition. The value part of the search condition indicates the operand against which the operator should be applied to the attribute. You will need to create a search condition using the attributes, objects and values provided in the query builder to search for claims.

 

Enter the following details to search for inventory items:

  • In the Provide a phrase describing the search condition, enter a name under which you want to save the search conditions for future searches.

  • If you want to view the results for a negation of the search conditions you create, then check the NOT check-box.

  • Select an option from the drop-down, All of the conditions are true or Atleast one of the conditions below is true.

  • In the Search Conditions section, click on the option. A new row will be added.

  • Use the drop-down to select an attribute for your search condition. For example, select Item Description from this drop-down.

  • Use the drop-down to select an object for your condition. The options in this drop-down depend on the attribute selected by you. For example, for the Item Description attribute, select starts with or contains or does not contain from this drop-down.

  • Now you need to enter a value in the text field. This is a free text field.

  • You can add multiple conditions for a more advanced search, by clicking on the option.

  • You can delete a condition by clicking on the corresponding to that row.

Once you set the search parameter, perform one of the following actions:

  • If you wish to save this search, then check the Save check-box. The search will be saved in the Search Folders section under the search phrase specified by you.

  • If you want to clear the options selected by you, click on the Reset button.

  • If you want to search for inventory items based on the condition you just created, click on the Execute button. The page refreshes to show the search results in a tabular format.

Edit Search

You can edit a search phrase and modify any of the search parameters.

 

To edit a search phrase,

  • Click on the button on the search result page.

  • The Edit Search Query page is shown. You can see the search conditions created by you here.

  • You can edit any parameter in this field.

  • To reset the data in this page, click Reset. All details entered by you will be reset to the default selections.

  • To execute the modified search conditions, click Execute. The search results will be shown in a tabular format.
